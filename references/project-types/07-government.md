# 项目类型信息架构手册 — 政务民生类

---

### 7.1 政务服务模式（Government Service Portal）
**适用类型**: 政务办事(一网通办), 行政审批, 电子证照

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页（服务入口网格） | 顶部搜索栏 + 宫格图标入口 + 公告轮播 + 热门事项推荐 | 全局搜索框、分类宫格导航、滚动公告横幅、"我要办"快捷入口卡片 |
| 办事事项列表 | 左侧分类树 + 右侧列表 + 顶部筛选栏 | 事项分类侧边栏、条件筛选器（主题/部门/级别）、分页列表、关键词高亮 |
| 事项详情页 | 顶部事项概要 + 标签页切换（材料/流程/收费标准） | 办理条件卡片、材料清单表格（含下载链接）、办理流程时间轴、常见问题折叠面板 |
| 在线申报（表单） | 分步表单向导 + 左侧进度指示器 | 动态表单引擎、附件上传组件、身份证/营业执照OCR识别、表单校验提示、暂存草稿按钮 |
| 进度查询 | 顶部查询入口 + 纵向时间线主体 | 办件编号查询框、流程时间线（节点状态图标）、当前环节高亮、预计完成时间提示 |
| 预约办事 | 日历选择 + 时段格子 + 确认弹窗 | 月历组件、可选时段网格（绿色/灰色状态）、预约确认对话框、短信提醒设置 |
| 个人中心 | 左侧导航菜单 + 右侧内容区 | 用户信息卡片、我的办件列表、电子证照夹、消息通知、收藏事项 |
| 咨询投诉 | 顶部入口分类 + 对话式表单 + 历史记录 | 咨询/投诉/建议分类Tab、富文本编辑器、附件上传、处理进度跟踪卡片 |
| 好差评 | 事项办理完成后弹出的评价卡片 | 星级评分、标签选择（态度/效率/便捷）、文字评价输入、匿名开关 |
| 智能问答助手 | 页面右下角悬浮气泡 + 展开式对话窗口 | AI对话机器人、热门问题推荐、转人工客服按钮、知识库检索结果卡片 |

**用户流程**: 用户进入首页 → 通过搜索或分类找到事项 → 查看事项详情了解材料清单 → 在线填写申报表单并上传材料 → 提交后获得办件编号 → 通过进度查询跟踪办理状态 → 办结后进行好差评
**类型差异**: 一网通办侧重跨部门联办事项的串联编排；行政审批强调多级审批流转与限时办结机制；电子证照聚焦证照的申领、亮证、验真与归集共享

### 7.2 税务申报模式（Tax Declaration）
**适用类型**: 个税申报, 企业税申报, 税务查询

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 税务首页 | 顶部身份切换（个人/企业）+ 待办提醒卡片 + 快捷入口 | 身份切换Toggle、申报日历卡片、待办事项红点提醒、税种快捷入口 |
| 税种选择 | 卡片网格展示各税种 + 申报周期标注 | 税种卡片（增值税/所得税/印花税等）、申报截止日期标签、已申报/未申报状态徽标 |
| 申报表填写 | 多步骤表单 + 左侧税目明细树 + 右侧计算面板 | 动态表单字段、自动计算引擎、专项附加扣除选择器、实时税额预览面板 |
| 收入明细 | 数据表格 + 顶部汇总统计条 | 收入来源分类饼图、明细数据表格（可展开行）、导入/手动录入切换 |
| 扣除项管理 | 列表 + 新增/编辑弹窗 | 专项附加扣除卡片列表、子女教育/房贷/租房等表单模板、佐证材料上传 |
| 缴税/退税 | 金额确认页 + 支付方式选择 | 应缴/应退税额大字展示、支付渠道选择（银行卡/三方支付）、退税银行卡绑定 |
| 申报记录查询 | 筛选栏 + 历史申报列表 | 年度/税种筛选器、申报记录表格（含状态标签）、详情展开查看、申报表PDF下载 |
| 发票管理 | 扫码/拍照入口 + 发票列表 | 发票OCR识别、发票验真按钮、进项/销项分类Tab、发票勾选确认 |
| 税务通知 | 消息列表 + 详情弹窗 | 政策公告卡片、个性化通知（催报/催缴）、政策原文折叠展示 |
| 纳税证明 | 申请入口 + 证明预览 | 完税证明申请表单、电子签章预览、PDF下载/分享按钮 |

**用户流程**: 用户登录切换身份 → 查看待申报税种 → 进入申报表填写 → 系统自动计算税额 → 确认提交申报 → 选择缴税或申请退税 → 查看历史记录与下载凭证
**类型差异**: 个税申报以年度汇算清缴为核心，强调专项附加扣除的便捷填报；企业税申报涉及多税种并行、财务报表附报与税控盘对接；税务查询侧重历史数据的多维度检索与可视化

### 7.3 社保/公积金模式（Social Security & Housing Fund）
**适用类型**: 社保查询, 公积金查询/提取, 社保卡管理

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 社保/公积金首页 | 账户概览卡片 + 功能宫格入口 | 账户余额大字展示、五险/公积金状态卡片、功能入口宫格、政策公告横幅 |
| 缴费明细 | 时间筛选 + 数据表格 + 趋势图 | 月度/年度筛选器、缴费明细表格（个人/单位分列）、缴费趋势折线图 |
| 参保信息 | 信息卡片纵向排列 | 参保单位、参保地、缴费基数、各险种状态标签（正常参保/停保） |
| 公积金提取 | 提取原因选择 + 金额输入 + 材料上传 | 提取场景选择器（购房/租房/离职等）、可提额度计算器、材料清单与上传、提取进度追踪 |
| 社保卡管理 | 卡片状态展示 + 操作按钮组 | 电子社保卡二维码、挂失/解挂按钮、补换卡申请表单、密码修改 |
| 医保账户 | 余额卡片 + 消费记录列表 | 医保余额展示、消费流水列表（医院/药店分类）、异地就医备案入口 |
| 养老金查询 | 累计信息卡片 + 预估计算器 | 累计缴费年限/金额展示、养老金预估计算器（滑块调节退休年龄）、待遇发放记录 |
| 转移接续 | 分步申请表单 + 进度追踪 | 转入/转出选择、原参保地与新参保地信息表单、办理进度时间轴 |
| 消息通知 | 消息列表分类Tab | 缴费到账通知、政策变动提醒、提取审批结果、年度对账单 |
| 个人权益记录 | 年度汇总报告式页面 | 年度缴费汇总、权益记录PDF预览与下载、电子签章 |

**用户流程**: 用户进入首页查看账户概览 → 按需查询缴费明细或参保信息 → 如需提取公积金选择场景并提交申请 → 管理社保卡状态 → 跟踪各项业务的办理进度
**类型差异**: 社保查询强调五险数据的分项展示与历史追溯；公积金突出提取/贷款场景的在线办理；社保卡管理聚焦实体卡与电子卡的生命周期管理

### 7.4 交通违章模式（Traffic Violation）
**适用类型**: 违章查询, 罚款缴纳, 驾照扣分

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 违章查询首页 | 车辆/驾照绑定卡片 + 快捷查询入口 | 已绑定车辆卡片列表、添加车辆表单（车牌号+行驶证OCR）、一键查询按钮 |
| 违章记录列表 | 筛选栏 + 违章记录卡片列表 | 时间/状态筛选器、违章卡片（时间/地点/违章行为/罚款金额）、未处理红点标记 |
| 违章详情 | 违章信息卡片 + 证据照片 + 操作按钮 | 违章地点地图定位、抓拍照片轮播、扣分/罚款明细、申诉入口按钮 |
| 罚款缴纳 | 待缴清单 + 支付确认页 | 可勾选违章列表、合计金额汇总、支付渠道选择、电子票据生成 |
| 驾照记分 | 记分周期仪表盘 + 记分明细 | 记分圆环进度图（12分制）、记分明细列表、清分日期倒计时、学法减分入口 |
| 学法减分 | 学习入口 + 考试界面 | 在线视频学习模块、答题考试界面、减分记录与剩余可减分值 |
| 车辆管理 | 车辆信息卡片 + 相关服务入口 | 车辆基本信息展示、年检到期提醒、保险到期提醒、环保标志申领 |
| 缴款记录 | 历史记录列表 + 电子票据 | 缴款历史表格（含状态）、电子票据预览与下载、退款申请入口 |
| 违章申诉 | 申诉表单 + 进度追踪 | 申诉原因选择、证据材料上传、申诉进度时间轴、结果通知 |
| 消息提醒 | 推送消息列表 | 新违章提醒、年检/保险到期提醒、缴款截止提醒、政策变动通知 |

**用户流程**: 用户绑定车辆 → 查询违章记录 → 查看违章详情与证据 → 勾选待缴违章进行罚款支付 → 查看驾照记分情况 → 如需可通过学法减分恢复记分 → 收到各类到期与违章提醒
**类型差异**: 违章查询聚焦多车辆管理与实时违章推送；罚款缴纳强调批量选择与多渠道支付体验；驾照扣分侧重记分周期管理与学法减分的在线学习考试

### 7.5 公益慈善模式（Charity & Public Welfare）
**适用类型**: 公益捐赠, 众筹慈善, 志愿者管理

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 公益首页 | 轮播Banner + 项目推荐卡片 + 数据概览 | 公益项目轮播、"推荐项目"卡片网格、累计捐赠金额/人数统计条、搜索入口 |
| 项目列表 | 分类筛选 + 项目卡片列表 | 项目分类Tab（教育/医疗/环保/救灾）、排序选择器、项目卡片（封面图+进度条+金额） |
| 项目详情 | 大图Banner + 故事描述 + 捐赠入口 | 项目故事富文本、目标金额进度条、捐赠人数与剩余天数、"立即捐赠"浮动按钮、项目动态时间轴 |
| 捐赠支付 | 金额选择 + 支付确认 | 预设金额按钮组、自定义金额输入、匿名捐赠开关、留言祝福输入、支付渠道选择 |
| 众筹页面 | 众筹进度仪表盘 + 支持者列表 | 众筹目标进度环形图、倒计时组件、支持者头像墙、分享传播按钮组 |
| 志愿者注册 | 分步注册表单 + 技能标签选择 | 个人信息表单、服务领域多选标签、可用时间段日历、资质证书上传 |
| 志愿活动列表 | 地图+列表双视图 | 活动卡片（时间/地点/内容）、地图标注视图切换、报名状态标签、服务时长统计 |
| 个人中心 | 公益档案卡片 + 功能菜单 | 捐赠记录列表、电子捐赠证书、志愿服务时长统计、税务抵扣凭证下载 |
| 项目进展 | 时间轴 + 图文更新 | 项目更新时间轴（含图文）、财务公示表格、受益人反馈卡片 |
| 透明度报告 | 数据看板 + 审计报告 | 资金流向桑基图、项目支出饼图、第三方审计报告PDF嵌入、公示日历 |

**用户流程**: 用户浏览公益首页 → 发现感兴趣的项目 → 查看详情与项目进展 → 选择金额完成捐赠 → 获得电子证书 → 持续关注项目进展与资金透明度报告 → 或注册成为志愿者参与活动
**类型差异**: 公益捐赠侧重单次/月捐的便捷支付与证书激励；众筹慈善强调社交传播与进度可视化的信任构建；志愿者管理聚焦注册、排班、服务时长记录与评价体系

### 7.6 投票/问卷模式（Voting & Survey）
**适用类型**: 在线投票, 问卷调查, 民意征集

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 投票/问卷首页 | 进行中的活动卡片列表 + 分类入口 | 活动卡片（标题/截止时间/参与人数）、分类Tab（投票/问卷/征集）、创建活动入口 |
| 投票页面 | 候选人/选项卡片 + 投票按钮 | 候选人卡片（照片+简介）、单选/多选投票组件、实时票数柱状图（投票后可见）、倒计时 |
| 问卷填写 | 分步问卷 + 顶部进度条 | 题目卡片（单选/多选/填空/量表/矩阵）、进度指示器、自动保存提示、提交确认 |
| 问卷设计器 | 左侧题型库 + 中间画布 + 右侧属性面板 | 拖拽式题型组件库、题目排序画布、逻辑跳转编辑器、预览模式切换、问卷主题皮肤选择 |
| 民意征集详情 | 背景说明 + 意见提交区 + 已有意见摘要 | 征集背景富文本、意见提交表单（支持富文本+附件）、意见汇总词云图、采纳情况标签 |
| 数据统计 | 图表仪表盘 + 数据表格 | 回收率统计卡片、每题分析图表（饼图/柱状图/条形图）、交叉分析矩阵、数据导出按钮 |
| 结果公示 | 结果摘要 + 详细图表 + 说明 | 投票结果排行榜、问卷分析摘要、民意采纳情况公示、PDF报告下载 |
| 发放管理 | 渠道选择 + 配额设置 + 进度追踪 | 发放渠道配置（链接/短信/邮件/二维码）、样本配额设置、回收进度进度条、催答按钮 |
| 模板库 | 分类模板卡片网格 | 模板分类Tab（满意度/需求调研/民主评议）、模板预览弹窗、一键使用按钮 |
| 我的问卷 | 管理列表 + 状态标签 | 问卷列表（草稿/进行中/已结束）、快捷操作菜单（编辑/复制/删除/导出） |

**用户流程**: 管理者从模板库或空白创建问卷/投票 → 设计题目与逻辑 → 设置发放渠道与配额 → 参与者通过链接/二维码进入填写 → 系统自动统计分析 → 管理者查看数据看板并导出报告 → 结果公示
**类型差异**: 在线投票强调候选展示与实时计票的公平透明；问卷调查侧重题型丰富性与逻辑跳转的灵活设计；民意征集聚焦开放式意见收集与采纳反馈闭环

### 7.7 公共信息模式（Public Information）
**适用类型**: 天气预报, 空气质量, 疫情信息, 地震预警

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 信息首页 | 当前状态大卡片 + 多维信息摘要 | 当前天气/空气质量/疫情大字数据卡片、预警横幅（红色/橙色/黄色）、快捷切换Tab |
| 天气预报详情 | 时间序列图表 + 生活指数卡片 | 逐时温度曲线图、7天/15天预报列表、生活指数网格（紫外线/穿衣/运动）、降水雷达图 |
| 空气质量详情 | 仪表盘 + 污染物分项 | AQI仪表盘（颜色编码）、PM2.5/PM10/SO2等分项柱状图、空气质量趋势折线图、健康建议卡片 |
| 疫情信息看板 | 数据总览 + 地图热力图 + 趋势图 | 新增/累计/治愈/死亡数据卡片、疫情地图热力分布、每日新增趋势折线图、风险等级区域列表 |
| 地震预警 | 预警弹窗 + 震中地图 + 信息列表 | 全屏预警弹窗（震级/震源/距离）、震中标注地图、历史地震列表、应急避险指南入口 |
| 预警通知设置 | 开关列表 + 阈值设置 | 预警类型开关（天气/地震/疫情）、推送阈值滑块、通知方式选择（App/短信/电话） |
| 历史数据查询 | 时间范围选择 + 数据图表 + 导出 | 日期范围选择器、历史数据折线图/柱状图、数据表格视图切换、CSV/PDF导出按钮 |
| 生活建议 | 卡片式建议列表 | 出行建议卡片、健康防护建议、运动建议、驾车建议（根据当前数据动态生成） |
| 地图总览 | 全屏地图 + 多图层叠加 | 地图底图、可切换图层（天气/空气/疫情/地震）、图例面板、定位到当前位置按钮 |
| 信息分享 | 生成分享卡片 + 社交分享 | 当前数据摘要分享卡片、生成海报按钮、社交平台分享按钮组、小程序码 |

**用户流程**: 用户打开首页查看当前状态概览 → 根据预警横幅判断紧急程度 → 进入详情页查看具体数据与趋势 → 查看生活建议做出出行决策 → 设置预警通知以便未来自动提醒 → 分享信息给家人朋友
