# 项目类型信息架构手册 — 制造业类

---

### 10.1 MES制造执行模式（Manufacturing Execution）
**适用类型**: MES系统, 车间排程, 工序追踪, 生产管理

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 数据卡片+图表 | 生产概览(计划/完成/良品率), 设备状态, 在制品, 告警 |
| 工单管理 | 表格+状态 | 工单列表, 状态(待排/生产中/已完成), 产品/数量/交期, 操作 |
| 排程 | 甘特图 | 资源(设备/产线)时间轴, 工单条, 拖拽调整, 冲突检测 |
| 设备管理 | 卡片+状态 | 设备列表, 运行状态, OEE, 维护计划, 故障记录 |
| 质量管理 | 检测记录 | 检验记录, 合格率, 不良品分析, SPC图表, CAPA |
| 物料管理 | 进销存 | 物料清单, 库存, 领料/退料, BOM, 缺料预警 |
| 报表分析 | 图表 | 产量/良品率/OEE趋势, 设备利用率, 成本分析, 导出 |
| 告警中心 | 列表 | 设备故障, 质量异常, 缺料, 交期预警, 处理记录 |

**用户流程**: 接收订单 → 排程 → 派工 → 生产执行 → 质检 → 入库 → 报表分析
**类型差异**: 离散制造重工单和BOM; 流程制造重配方和批次; 重工单重调度; 重OEE重设备效率

---

### 10.2 质量管理(QMS)模式（Quality Management）
**适用类型**: 来料检验, 过程质检, 不良品追溯, 质量审核

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 图表+指标 | 质量KPI(合格率/PPM/客诉), 趋势, 待处理项 |
| 检验管理 | 表格+表单 | 检验计划, 检验记录, 判定(合格/不合格/让步), 图片 |
| 不良品管理 | 列表+分析 | 不良品列表, 不良类型, 根因分析(鱼骨图/5Why), 处置 |
| CAPA | 流程+表单 | 纠正/预防措施, 责任人, 截止日期, 验证, 关闭 |
| 审核管理 | 日历+列表 | 审核计划, 审核记录, 不符合项, 整改 |
| 文档管理 | 文件夹+列表 | 质量手册, 程序文件, 作业指导书, 版本控制 |
| SPC | 控制图 | 控制图(Xbar-R/P/U), 过程能力(Cpk), 异常检测 |

**用户流程**: 来料检验 → 过程检验 → 成品检验 → 不良分析 → CAPA → 持续改进
**类型差异**: 来料重抽样和标准; 过程重SPC和实时; 成品重全检和认证; 审核重计划和整改

---

### 10.3 供应链协同(SCM)模式（Supply Chain）
**适用类型**: 供应商协同, 采购对账, 交期管理, 供应商评估

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 数据+图表 | 采购概览, 供应商绩效, 交期达成率, 成本趋势 |
| 供应商管理 | 列表+详情 | 供应商列表, 资质, 评级, 合作历史, 联系 |
| 采购订单 | 表格+流程 | PO列表, 审批流, 供应商确认, 到货, 对账 |
| 交期管理 | 甘特图+列表 | 订单交期, 物流状态, 延迟预警, 催货 |
| 对账 | 表格 | 采购/收货/发票三方匹配, 差异标记, 确认 |
| 评估 | 雷达图+表格 | 质量/交期/服务/价格评分, 趋势, 对比 |

**用户流程**: 需求 → 询价/比价 → 下单 → 跟踪交期 → 收货检验 → 对账付款 → 评估
**类型差异**: 协同重信息共享; 采购重流程和审批; 交期重预警和催货; 评估重多维和趋势

---

### 10.4 PLM产品生命周期模式（Product Lifecycle）
**适用类型**: 产品研发管理, BOM管理, 变更协同, 设计评审

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 项目概览 | 在研项目, 里程碑, 问题跟踪, 变更请求 |
| 项目管理 | 甘特图+看板 | 项目计划, 阶段门, 交付物, 团队, 评审 |
| BOM管理 | 树形+表格 | 产品BOM树, 多层展开, 版本对比, 替代件, 成本汇总 |
| 文档管理 | 文件夹+版本 | 设计文档, 图纸, 规格书, 版本历史, 签审状态 |
| 变更管理 | 流程+表单 | ECN/ECO流程, 影响分析, 审批, 执行跟踪 |
| 评审 | 列表+表单 | 评审计划, 评审记录, 问题列表, 决议, 签审 |

**用户流程**: 概念 → 设计 → 评审 → 试产 → 量产 → 变更 → 退市
**类型差异**: 研发重创新和验证; BOM重多层和成本; 变更重影响和审批; 评审重问题和签审

---

### 10.5 CAD/CAE/CAM模式（Industrial Design）
**适用类型**: 三维设计(CAD), 仿真分析(CAE), 数控编程(CAM), 工业设计

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 工作台 | 3D视图+工具栏 | 3D建模视图, 特征树(左侧), 属性面板(右侧), 工具栏 |
| 装配 | 3D+约束 | 装配视图, 约束管理, 干涉检查, BOM |
| 工程图 | 2D图纸 | 多视图投影, 尺寸标注, 公差, 明细表, 图框 |
| 仿真 | 设置+结果 | 网格划分, 边界条件, 求解, 应力/变形/热分析结果 |
| CAM编程 | 3D+刀路 | 工件设置, 刀具选择, 刀路生成, 仿真验证, G代码 |
| 协作 | 列表+版本 | 文件版本, 评论, 标记, 权限, 分享 |

**用户流程**: 建模 → 装配 → 出图 → 仿真验证 → CAM编程 → 加工
**类型差异**: CAD重建模和参数化; CAE重网格和求解; CAM重刀路和效率; 工业设计重曲面和美学

---

### 10.6 仓储管理(WMS)模式（Warehouse Management）
**适用类型**: 仓库管理, 入库出库, 库位管理, 拣货优化

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 数据+地图 | 库存概览, 入库/出库统计, 库位利用率, 告警 |
| 入库 | 表单+列表 | 收货单, 质检, 上架指引, 确认 |
| 出库 | 列表+指引 | 拣货单, 拣货路径, 复核, 打包, 发货 |
| 库位管理 | 仓库地图+网格 | 库位布局图, 占用状态, 货品信息, 调整 |
| 盘点 | 表单+列表 | 盘点计划, 盘点单, 差异分析, 调整 |
| 库存报表 | 图表 | 库存周转, 呆滞分析, ABC分类, 安全库存 |

**用户流程**: 收货 → 质检 → 上架 → 存储 → 拣货 → 复核 → 发货 → 盘点
**类型差异**: 电商仓重效率和波次; 冷链重温控; 危化品重安全; 自动化重设备调度

---

### 10.7 餐饮供应链模式（Food Supply Chain）
**适用类型**: 食材采购, 库存管理, 供应商管理, 中央厨房

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 数据+告警 | 采购概览, 库存预警, 供应商状态, 成本分析 |
| 采购 | 表单+列表 | 采购需求, 询价, 订单, 到货验收 |
| 库存 | 表格+预警 | 食材库存, 保质期, 先进先出, 损耗, 预警 |
| 供应商 | 列表+评估 | 供应商列表, 资质, 评价, 价格对比 |
| 中央厨房 | 排程+配方 | 生产计划, 标准配方, 成本核算, 配送 |
| 报表 | 图表 | 采购成本, 库存周转, 损耗分析, 毛利 |

**用户流程**: 需求预测 → 采购 → 验收 → 入库 → 领料/配送 → 成本核算
**类型差异**: 采购重价格和供应商; 库存重保质期和损耗; 中央厨房重配方和配送

---

### 10.8 安全生产模式（Safety Production）
**适用类型**: 隐患排查, 安全培训, 作业许可, 风险管控

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 数据+告警 | 安全指标, 隐患统计, 培训完成率, 事故记录 |
| 隐患排查 | 列表+表单 | 隐患列表, 拍照上报, 整改跟踪, 验收关闭 |
| 培训管理 | 列表+进度 | 培训计划, 课程, 考试, 证书, 完成率 |
| 作业许可 | 流程+表单 | 动火/高空/受限空间许可, 审批, 安全措施, 签字 |
| 风险管控 | 矩阵+地图 | 风险点清单, 风险矩阵(概率×后果), 管控措施, 四色图 |
| 事故管理 | 流程+报告 | 事故报告, 调查, 根因分析, 整改措施, 统计 |

**用户流程**: 识别风险 → 排查隐患 → 培训教育 → 许可审批 → 事故处理 → 持续改进
**类型差异**: 隐患重发现和整改; 培训重课程和考核; 许可重审批和安全; 风险重评估和管控

---

### 10.9 应急管理/环保模式（Emergency & Environmental）
**适用类型**: 应急预案, 碳排放管理, 环保监测, 危化品管理

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 应急指挥 | 地图+面板 | 事件地图, 资源分布, 人员调度, 通信, 决策支持 |
| 预案管理 | 文档+流程 | 预案库, 响应级别, 处置流程, 演练记录 |
| 碳排放 | 仪表盘+报表 | 排放量, 各源排放, 减排措施, 碳交易, 报告 |
| 环保监测 | 图表+地图 | 排放数据, 达标分析, 超标告警, 监测站地图 |
| 合规管理 | 列表+日历 | 许可证, 报告截止日, 检查记录, 整改 |

**用户流程**: 监测 → 发现异常 → 启动预案 → 指挥调度 → 事后总结 → 合规报告
**类型差异**: 应急重响应和调度; 碳重核算和交易; 环保重达标和报告; 危化品重存储和运输
