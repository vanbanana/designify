# 12 — 交通物流类

> 覆盖出行打车、导航、公共交通、共享单车、物流货运、快递、仓储、跨境物流、停车、拼车等场景。

---

### 12.1 打车出行（Ride-hailing）
**适用类型**: 网约车平台（滴滴/Uber/Lyft）、出租车调度、企业用车

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 叫车首页 | 全屏地图+底部面板 | 地图（当前位置+车辆标注）、目的地输入框、车型选择（快车/专车/拼车/豪华）、预估价格、叫车按钮 |
| 等待接驾 | 地图+状态卡片 | 司机信息（头像/姓名/车牌/车型/评分）、预计到达倒计时、司机位置实时追踪、联系/取消按钮 |
| 行程中 | 全屏地图+底部信息 | 路线实时绘制、剩余距离/时间、费用实时更新、安全工具（分享行程/紧急求助）、途经点添加 |
| 行程结束/支付 | 单栏+费用明细 | 费用明细（起步价+里程+时长+附加费）、支付方式选择、评价司机（星级+标签+文字）、小费 |
| 行程历史 | 列表 | 历史行程卡片（日期/路线/费用/状态）、重新叫车、开发票、投诉 |
| 优惠券/钱包 | 列表+卡片 | 可用优惠券列表、余额、充值、消费记录、过期提醒 |
| 常用地址 | 列表+编辑 | 家/公司/收藏地址、地址搜索+地图选点、编辑/删除 |
| 安全中心 | 卡片+设置 | 行程分享设置、紧急联系人、录音录像设置、安全驾驶报告 |
| 司机端-接单 | 地图+弹窗 | 附近订单气泡、接单/拒单、导航到上车点、乘客信息 |
| 司机端-收入 | 仪表盘 | 今日/本周收入、订单数、在线时长、服务分、奖励任务 |
| 登录/注册 | 单栏居中 | 手机号登录、微信/支付宝、司机入驻（驾照+行驶证+背景审查） |

**用户流程**: 
- **乘客**: 输入目的地→选车型→叫车→等待接驾→行程中→到达支付→评价
- **司机**: 上线→接单→导航到乘客→开始行程→到达→收款→查看收入
**类型差异**: 网约车重算法派单和动态定价; 出租车重调度效率; 企业用车重审批和报销

---

### 12.2 导航地图（Navigation/Maps）
**适用类型**: 地图导航App（高德/百度/Google Maps）、车载导航、户外导航

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 地图首页 | 全屏地图+浮动控件 | 地图（2D/3D/卫星视图切换）、定位按钮、搜索框、图层按钮、方向指南针 |
| 搜索/探索 | 搜索框+结果列表 | POI搜索、周边推荐（美食/酒店/加油站/停车场）、语音搜索、搜索历史、热门搜索 |
| 路线规划 | 地图+路线面板 | 起终点输入、出行方式切换（驾车/公交/步行/骑行）、多路线对比（时间/距离/红绿灯）、途经点 |
| 导航中 | 全屏地图+HUD | 实时路线指引、转向箭头+距离、限速提醒、路况颜色（红/黄/绿）、预计到达时间、语音播报 |
| POI详情页 | 图文+卡片 | 地点名称+分类+评分、图片、营业时间、电话/地址、用户评价、路线规划按钮、收藏 |
| 实时路况 | 地图+图层 | 拥堵热力图、事故/施工标注、限行区域、测速摄像头位置 |
| 公交/地铁 | 时间线+列表 | 附近站点、到站倒计时、线路查询、换乘方案、首末班车时间 |
| 收藏/足迹 | 列表+地图标注 | 收藏地点分组、历史访问、去过的地方地图标注 |
| 离线地图 | 列表+下载 | 城市/区域列表、已下载管理、存储空间、自动更新设置 |
| 设置 | 分组表单 | 地图偏好（单位/语言）、导航语音、避开偏好（高速/收费站）、隐私设置 |

**用户流程**: 搜索目的地→查看POI详情→规划路线→开始导航→到达→收藏/评价
**类型差异**: 车载导航重大屏适配和语音交互; 户外导航重等高线和离线; 公交导航重实时到站

---

### 12.3 公交/地铁（Public Transit）
**适用类型**: 公交查询App、地铁官方App、城市一卡通

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 卡片+附近站点 | 附近公交/地铁站列表、实时到站倒计时、常用线路快捷入口、出行提醒横幅 |
| 线路查询 | 地图+时间线 | 线路搜索、线路图展示、站点列表、首末班车时间、换乘标注 |
| 换乘规划 | 表单+方案列表 | 起终点输入、多方案对比（时间/换乘次数/步行距离）、步行导航衔接 |
| 到站提醒 | 列表+倒计时 | 关注站点实时到站信息、到站震动/铃声提醒设置 |
| 乘车码 | 居中二维码 | NFC/二维码乘车码、刷新、切换城市、余额/扣费记录 |
| 站点详情 | 信息+地图 | 站点出入口地图、无障碍设施标注、周边POI、首末班车 |
| 线路详情 | 站点时间线 | 全线站点列表、实时车辆位置标注、到站预测、拥挤度指示 |
| 失物/公告 | 列表 | 失物招领列表、运营公告（延误/施工/调整）、FAQ |
| 票务/充值 | 表单+记录 | 一卡通充值、电子票购买、充值记录、消费明细 |
| 登录/城市 | 单栏居中 | 城市选择（自动定位+手动）、账号登录、乘车码绑定 |

**用户流程**: 打开App→查看附近站点/到站时间→规划换乘→刷码乘车→到站提醒→出站
**类型差异**: 公交重实时到站和线路覆盖; 地铁重换乘方案和拥挤度; 一卡通重充值和消费

### 12.4 共享单车/电动车（Bike/Scooter Sharing）
**适用类型**: 共享单车（美团单车/哈啰）、共享电动车、共享滑板车

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 全屏地图+底部面板 | 地图（附近车辆标注）、扫码开锁按钮、车辆类型切换（单车/电动车）、预估费用 |
| 扫码/开锁 | 扫码界面+状态 | 相机扫码框、手动输入车辆编号、开锁动画、蓝牙连接状态 |
| 骑行中 | 全屏地图+骑行数据 | 骑行时间、骑行距离、实时费用、限速提醒（电动车）、临时锁车按钮 |
| 还车/结算 | 地图+费用+评价 | 还车区域检测（运营区内/外）、停车拍照、费用明细、满意度评价 |
| 骑行历史 | 列表 | 历史骑行卡片（日期/路线/距离/费用）、骑行轨迹回放 |
| 月卡/套餐 | 卡片+购买 | 月卡/季卡/次卡套餐对比、购买、使用次数统计、到期提醒 |
| 运营区/禁停 | 地图+区域标注 | 运营区边界、禁停区标注、调度费说明、超区提醒 |
| 报修/反馈 | 表单+拍照 | 故障类型选择（刹车/链条/座椅等）、拍照上传、车辆编号、位置自动获取 |
| 钱包/余额 | 列表+充值 | 余额、押金、充值、消费记录、退款申请 |
| 登录/注册 | 单栏居中 | 手机号/微信、实名认证、押金/信用免押（芝麻信用） |

**用户流程**: 打开App→扫码开锁→骑行→到达目的地→还车（拍照确认）→自动扣费→评价
**类型差异**: 单车重骑行体验和月卡; 电动车重电量和限速; 滑板车重安全提醒

---

### 12.5 物流货运（Freight/Logistics）
**适用类型**: 货运平台（满帮/货拉拉）、合同物流、零担快运、冷链物流

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 发货首页 | 表单+地图 | 发货地/收货地输入、货物信息（品名/重量/体积/件数）、车型选择、预估运费 |
| 找车/找货 | 列表+筛选 | 车源/货源列表、筛选（线路/车型/载重）、匹配度排序、在线询价 |
| 运单详情 | 时间线+地图 | 运单状态节点（接单/装货/在途/签收）、车辆位置实时追踪、司机信息、电子回单 |
| 在途监控 | 地图+数据面板 | 所有在途车辆地图标注、ETA预测、偏航告警、温度监控（冷链）、超时预警 |
| 运费结算 | 表格+图表 | 运费明细、月结账单、对账管理、发票申请、支付方式 |
| 司机管理 | 列表+详情 | 司机信息（驾照/车辆/资质）、评分、历史运单、在线状态 |
| 车辆管理 | 台账+状态 | 车辆档案（牌照/载重/保险）、GPS设备、年检/保险到期提醒、维保记录 |
| 合同管理 | 列表+文档 | 运输合同模板、电子签约、合同台账、续约提醒 |
| 调度中心 | 看板+地图 | 待派单/已派单/在途看板、车辆可用状态、智能推荐调度方案 |
| 报表分析 | 多图表 | 运输量趋势、时效达标率、成本分析、线路优化建议 |
| 异常处理 | 列表+工单 | 货损货差记录、异常照片、理赔申请、处理进度 |
| 登录/角色 | 单栏居中 | 货主/司机/承运商/调度员角色、企业认证 |

**用户流程**: 
- **货主**: 发布货源→选择司机→跟踪在途→签收确认→结算运费
- **司机**: 浏览货源→接单→导航装货→在途运输→送达签收→收款
**类型差异**: 整车重线路匹配和运价; 零担重拼车和分拣; 冷链重温控和合规; 合同物流重KPI和报表

### 12.6 快递查询（Express Delivery Tracking）
**适用类型**: 快递查询App、快递柜、驿站管理、快递公司官方App

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 搜索+卡片列表 | 运单号搜索/扫码、在途包裹卡片列表（预计到达倒计时）、已签收包裹 |
| 包裹详情 | 时间线+地图 | 物流节点时间线（揽收→中转→派送→签收）、当前位置地图、快递员信息、预计送达 |
| 包裹地图 | 全屏地图+标注 | 包裹实时位置、经过城市标注、分拣中心标注、最后更新位置 |
| 寄件 | 表单+预估 | 寄件人/收件人信息、物品类型、重量/体积、快递公司选择、费用预估、预约上门取件 |
| 快递柜 | 地图+取件码 | 附近快递柜位置、可用格口、取件码输入、远程开柜、超时提醒 |
| 驿站管理 | 列表+操作 | 入库扫码、取件通知发送、滞留件提醒、出库确认、库存盘点 |
| 物流消息 | 通知列表 | 派送通知、签收确认、异常通知（地址不详/拒收）、快递员留言 |
| 收寄记录 | 列表+筛选 | 历史收发记录、按时间/快递公司筛选、重新寄件、开发票 |
| 价格查询 | 表单+对比 | 起终点+重量输入、多快递公司价格对比、时效对比 |
| 投诉/理赔 | 表单+工单 | 投诉类型选择、运单号关联、拍照举证、处理进度跟踪 |
| 登录 | 单栏居中 | 手机号/微信、绑定常用地址、快递公司账号 |

**用户流程**: 
- **收件人**: 查看在途包裹→跟踪物流→收到取件通知→快递柜/驿站取件→确认签收
- **寄件人**: 填写寄件信息→选择快递公司→预约取件→跟踪→确认签收
**类型差异**: 查询App重聚合多家; 快递公司App重寄件和服务; 快递柜重格口管理; 驿站重批量出入库

### 12.7 仓储管理/WMS（Warehouse Management）
**适用类型**: 电商仓库、第三方仓储、保税仓、冷链仓库

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仓库总览 | 仪表盘+平面图 | 库存总量/金额、今日入出库单量、库位利用率、仓库平面图（热力图显示占用率） |
| 入库管理 | 列表+表单 | 入库单列表（待收/已收/异常）、收货验收表单、上架推荐库位、批次管理 |
| 出库管理 | 列表+波次 | 出库单列表、波次策略（按订单/区域/承运商）、拣货任务分配、复核打包 |
| 库存管理 | 表格+筛选 | SKU库存列表、库位查询、批次/效期管理、库存盘点、安全库存预警 |
| 库位管理 | 平面图+树 | 仓库-区域-货架-库位树形结构、库位状态（空闲/占用/锁定）、库位调整 |
| 盘点管理 | 表单+差异表 | 盘点计划（全盘/抽盘）、PDA扫码录入、盘点差异表、盈亏处理 |
| 拣货路径 | 平面图+路径线 | 拣货任务列表、最优路径计算、PDA指引、拣货确认 |
| 报表分析 | 多图表 | 库存周转率、出入库趋势、SKU动销分析、库龄分析、呆滞库存预警 |
| 设备管理 | 列表+状态 | 叉车/PDA/输送线状态、维保计划、故障记录 |
| 集成设置 | 配置表单 | ERP/OMS系统对接、快递公司对接、条码/RFID配置、打印模板 |
| 登录/权限 | 单栏居中 | 仓库切换、角色（仓管/拣货/收货/主管）、PDA扫码登录 |

**用户流程**: 收货入库（验收→上架）→库存管理（盘点→调拨）→出库（波次→拣货→复核→发货）→报表分析
**类型差异**: 电商仓重SKU多和时效; 保税仓重海关监管; 冷链重温控; 三方仓重多货主隔离

### 12.8 跨境物流（Cross-border Logistics）
**适用类型**: 跨境电商物流、国际货代、海外仓、保税物流

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 跨境首页 | 仪表盘+地图 | 全球在途货物地图、今日发运/到达统计、各线路时效概览、异常告警 |
| 发货管理 | 多步表单 | 发货信息（品名/HS编码/申报价值）、贸易条款（FOB/CIF/DDP）、运输方式选择（空运/海运/铁路/快递）、报关文件上传 |
| 物流跟踪 | 时间线+世界地图 | 全程物流节点（国内揽收→报关→海运/空运→目的港清关→末端配送）、ETA预测、里程碑进度条 |
| 报关管理 | 表单+状态 | 报关单录入、HS编码查询、税费预估、报关状态跟踪、放行通知 |
| 海外仓管理 | 仪表盘+库存 | 各海外仓库存总览、入库/出库记录、FBA转运、库龄分析、仓储费计算 |
| 费用管理 | 表格+图表 | 运费/关税/仓储费/操作费明细、多币种结算、汇率管理、对账单 |
| 渠道管理 | 对比表+设置 | 物流渠道对比（时效/价格/覆盖）、渠道启用/禁用、运费模板设置 |
| 合规管理 | 列表+文档 | 出口管制清单筛查、制裁国家检查、原产地证明、MSDS管理 |
| 退货管理 | 流程+表单 | 退货申请、海外仓收货、质检、重新入库/销毁决策 |
| 报表 | 多图表 | 各线路时效分析、物流成本占比、清关通过率、妥投率 |
| 登录/多语言 | 单栏居中 | 多语言切换、多币种、角色（发货人/货代/报关行/海外仓） |

**用户流程**: 创建发货单→选择渠道→准备报关文件→国内揽收→报关→国际运输→目的国清关→末端配送→签收
**类型差异**: 快递模式重时效; 海运重成本和体积; FBA重亚马逊入仓规则; 海外仓重本地配送

---

### 12.9 停车管理（Parking Management）
**适用类型**: 智慧停车App、停车场管理系统、路侧停车、停车诱导

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 找车位首页 | 全屏地图+底部面板 | 附近停车场标注（空位数/价格/距离）、路侧空位标注、导航到停车场按钮 |
| 停车场详情 | 信息卡片+地图 | 停车场名称、总车位/空位数（实时更新）、收费标准、营业时间、出入口位置地图 |
| 导航入场 | 地图+路线 | 场内导航、空闲区域指引、车位级导航（室内定位） |
| 缴费/出场 | 费用+支付 | 停车时长、费用明细、优惠券/会员折扣、无感支付/扫码支付、电子发票 |
| 月卡/长租 | 卡片+表单 | 月卡套餐对比、购买/续费、车辆绑定、到期提醒 |
| 停车记录 | 列表 | 历史停车记录（时间/地点/时长/费用）、开发票 |
| 反向寻车 | 地图+搜索 | 输入车牌号、车辆位置标注、导航到车辆路线 |
| 管理后台-监控 | 大屏+数据 | 车位占用率实时大屏、车流进出统计、收入看板、设备状态 |
| 管理后台-设备 | 列表+状态 | 道闸/摄像头/地磁/诱导屏设备台账、在线状态、故障告警 |
| 管理后台-费率 | 表单+规则 | 收费规则配置（时段/车型/首小时+超时）、节假日特殊费率 |
| 登录 | 单栏居中 | 车主/管理员角色、车牌号绑定、无感支付签约 |

**用户流程**: 
- **车主**: 找车位→导航到场→入场停车→办事→反向寻车→缴费出场
- **管理者**: 监控车位占用→处理异常→调整费率→查看收入报表
**类型差异**: 商业停车场重车位引导; 路侧停车重巡检和缴费; 停车诱导重城市级信息聚合

### 12.10 拼车/通勤（Carpooling/Commuting）
**适用类型**: 拼车平台、企业班车、顺风车、通勤App

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 地图+路线列表 | 附近拼车路线、常用通勤路线快捷入口、发布行程按钮、实时车辆位置 |
| 发布行程 | 表单+地图 | 出发地/目的地（地图选点）、出发时间、途经点、可载人数、费用分摊设置 |
| 搜索/匹配 | 列表+地图 | 匹配行程列表（时间/路线/匹配度）、车主信息（评分/历史）、一键预约 |
| 行程详情 | 时间线+地图 | 路线地图、途经站点、同乘人员信息、费用分摊明细、聊天入口 |
| 通勤日历 | 日历+行程 | 日历视图显示每日行程、重复行程设置、请假/取消标记 |
| 费用管理 | 表格+结算 | 分摊费用明细、月度结算、支付方式、企业补贴 |
| 评价/信用 | 列表+评分 | 互评记录、信用分、守时统计、爽约记录 |
| 企业班车 | 线路+时刻表 | 班车线路图、时刻表、实时位置追踪、上下车站点、座位预约 |
| 社区/公告 | 列表+卡片 | 拼车社区帖子、通勤攻略、线路讨论、失物招领 |
| 登录/注册 | 单栏居中 | 手机号/微信、企业认证（企业班车）、车辆认证（顺风车） |

**用户流程**: 
- **车主**: 发布行程→等待匹配→确认乘客→出发→到达→费用结算
- **乘客**: 搜索匹配行程→预约→等待上车→行程中→到达→支付+评价
**类型差异**: 顺风车重匹配和信任; 企业班车重线路和时刻; 通勤App重日历和重复性
