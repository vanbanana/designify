# 11 — 建筑能源类

> 覆盖建筑工程、建材交易、能源管理、光伏监控、加油站、物业管理等场景。

---

### 11.1 工程项目管理（Construction Project Management）
**适用类型**: 建筑工程管理、装修工程、市政工程、基础设施建设、EPC总包

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 项目总览仪表盘 | 仪表盘+卡片 | 项目数量/状态统计卡片、进度甘特图、里程碑时间线、预警看板（延期/超支/安全） |
| 项目列表 | 表格+筛选栏 | 项目名称/编号、状态标签（规划/施工/竣工）、进度条、负责人、投资金额、筛选（地区/类型/状态） |
| 项目详情 | Tab分区+时间线 | 基本信息卡片、施工进度甘特图、里程碑列表、参与方信息、变更记录、质量安全日志 |
| 施工进度管理 | 甘特图+看板 | WBS任务分解、任务依赖关系线、关键路径高亮、拖拽调整、完成百分比、资源分配面板 |
| 图纸/文档管理 | 文件夹树+预览 | CAD图纸在线预览、版本历史、批注标记、文档分类（设计/施工/竣工）、下载/分享 |
| 质量检查 | 表单+列表 | 检查模板选择、拍照上传+标注、问题定位（楼层/区域）、整改通知推送、复查确认 |
| 安全管理 | 卡片+时间线 | 安全日报统计、隐患排查列表、事故记录、安全培训签到、应急预案入口 |
| 材料/设备管理 | 表格+出入库记录 | 材料清单、库存数量、进场/退场记录、供应商信息、设备台账、维保计划 |
| 成本/预算管理 | 图表+表格 | 预算分解饼图、实际支出折线图、偏差分析、变更签证费用、支付进度 |
| 人员管理 | 表格+考勤 | 工人花名册、考勤打卡记录、工资结算、资质证书管理、安全教育记录 |
| 报告/月报 | 表单+图表 | 月报模板生成、进度/质量/安全/成本四维图表、PDF导出、审批流转 |
| 登录/权限 | 单栏居中 | 企业账号登录、角色权限（项目经理/监理/施工方/业主）、组织架构选择 |

**用户流程**: 登录→项目总览→进入项目详情→查看进度/质量/安全→处理整改/审批→生成报告
**类型差异**: 建筑工程重施工进度和多方协同; 装修工程重设计方案和材料选择; 市政工程重审批流程; EPC重设计-采购-施工全链条

### 11.2 建材采购（Building Materials Marketplace）
**适用类型**: 建材商城、钢材交易平台、水泥/混凝土订购、装修材料电商

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | F型+分类导航 | 搜索框、建材分类入口（钢材/水泥/木材/涂料/五金）、品牌专区、价格看板、促销活动 |
| 建材列表页 | 网格/列表可切换 | 筛选（品类/品牌/规格/产地）、排序（价格/销量）、产品卡片（图+名+规格+单价）、批量询价 |
| 建材详情页 | 双栏（图+信息） | 产品图片、规格参数表（型号/尺寸/重量/强度等级）、价格阶梯（量大优惠）、库存/配送范围、样品申请、质检报告下载 |
| 询价/比价 | 表单+表格 | 需求表单（品类+数量+规格+送达地址）、多供应商报价对比表、历史价格走势 |
| 购物车/订单 | 单栏列表 | 商品清单（含吨/方/件等多单位）、运费计算（按重量/距离）、合同生成、付款 |
| 物流跟踪 | 时间线+地图 | 运输状态节点、车辆位置地图、预计到达时间、签收确认 |
| 供应商管理 | 列表+详情 | 供应商资质、历史合作记录、评分评级、黑名单管理 |
| 合同管理 | 列表+文档预览 | 电子合同模板、在线签署、合同台账、履约跟踪 |
| 质检报告 | 列表+PDF预览 | 出厂检验报告、第三方检测证书、环保认证 |
| 财务结算 | 仪表盘+表格 | 应付/应收款、账期管理、发票管理、对账单 |
| 项目用料管理 | 表格+图表 | 项目用料计划、实际消耗对比、库存预警、补货提醒 |
| 登录/注册 | 单栏居中 | 企业注册（营业执照）、个人注册、供应商入驻 |

**用户流程**: 搜索建材→筛选比较→查看详情/询价→加入购物车→下单（生成合同）→跟踪物流→验收→结算
**类型差异**: 钢材平台重规格参数和期货价格; 水泥/混凝土重配送时效和搅拌站距离; 装修材料重环保认证和色卡

### 11.3 能耗管理（Energy Management System）
**适用类型**: 建筑能耗监测、工厂能源管理、碳排放管理、智慧楼宇

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 能耗总览仪表盘 | 仪表盘+地图 | 总能耗数字（今日/本月/本年）、能耗趋势折线图、区域能耗热力图、同比环比对比、碳排放量 |
| 实时监控 | 地图+数据面板 | 建筑/楼层/区域平面图、各点位实时能耗数字、设备运行状态指示灯、异常告警弹窗 |
| 能耗分析 | 多图表+筛选 | 分类能耗饼图（电/水/气/热）、分项柱状图（照明/空调/动力）、时段分布、峰谷分析 |
| 用能报告 | 图表+表格 | 日/周/月/年报告自动生成、对标分析（单位面积能耗 vs 行业标准）、节能建议、PDF导出 |
| 设备管理 | 列表+详情 | 计量设备台账（电表/水表/气表）、在线状态、数据采集频率、校准记录 |
| 告警管理 | 列表+规则配置 | 超限告警记录、告警规则设置（阈值/持续时间/通知方式）、告警趋势统计 |
| 碳管理 | 仪表盘+时间线 | 碳排放总量、Scope1/2/3分项、碳配额管理、碳交易记录、碳中和路径规划 |
| 节能优化 | 对比+建议卡片 | 节能空间分析、设备效率排名、优化策略建议（AI推荐）、预期节省金额 |
| 计费管理 | 表格+账单 | 租户/部门用能分摊、费率设置、账单生成、缴费状态 |
| 系统设置 | 分组表单 | 建筑信息配置、区域划分、设备绑定、数据源接入、用户权限 |
| 登录/权限 | 单栏居中 | 企业登录、角色（管理员/运维/财务/租户）、多建筑切换 |

**用户流程**: 登录→查看能耗总览→发现异常告警→进入分析页定位原因→执行节能策略→生成报告→考核碳排放
**类型差异**: 建筑能耗重楼层/租户分摊; 工厂重产线/设备能效; 碳管理重Scope分类和碳交易

### 11.4 光伏监控（Solar/PV Monitoring）
**适用类型**: 光伏电站监控、分布式光伏管理、户用光伏App、储能系统

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 电站总览 | 仪表盘+地图 | 今日发电量、累计发电量、实时功率、收益统计、CO2减排量、电站分布地图 |
| 实时监控 | 数据面板+图表 | 逆变器实时数据（功率/电压/电流/温度）、组串状态、电网连接状态、气象站数据 |
| 发电分析 | 多图表+时间选择 | 日/月/年发电曲线、PR（Performance Ratio）趋势、辐照度vs发电量对比、发电量排名 |
| 告警管理 | 列表+设备定位 | 故障告警列表（逆变器离线/组串失配/电网异常）、故障定位在电站平面图上、工单派发 |
| 收益管理 | 图表+表格 | 上网电费收入、补贴收入、自用节省、收益趋势、账单明细 |
| 设备管理 | 列表+详情 | 逆变器/组串/汇流箱/变压器台账、运行状态、维保记录、固件升级 |
| 储能管理 | 仪表盘+曲线 | 电池SOC/SOH、充放电功率曲线、峰谷套利策略设置、备用电源模式 |
| 运维管理 | 工单列表+日历 | 巡检计划日历、工单流转、清洗记录、除草记录、备品备件库存 |
| 报表中心 | 模板+导出 | 日报/月报/年报模板、自定义报表、对标分析（PR/等效利用小时数）、PDF/Excel导出 |
| 电站地图 | 全屏地图+标注 | 所有电站标注（装机容量/今日发电/状态）、点击展开详情卡片、导航到电站 |
| 登录/多电站 | 单栏居中 | 账号登录、多电站切换、角色（站长/运维/业主/投资方） |

**用户流程**: 查看发电总览→检查实时状态→处理告警→分析发电效率→查看收益→安排运维
**类型差异**: 大型地面电站重设备管理和运维效率; 分布式重多站点聚合; 户用重收益展示和简洁度; 储能重充放电策略

### 11.5 加油站管理（Gas Station Management）
**适用类型**: 加油站运营、连锁油站管理、充电桩+油站综合体

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 油站总览 | 仪表盘+卡片 | 今日销售额/升数、油品库存、车流量、会员数、实时油价看板 |
| 油品库存管理 | 表格+液位图 | 各油罐液位实时数据（92#/95#/98#/0#柴油）、库存预警、进油记录、损耗统计 |
| 销售分析 | 多图表 | 时段销量分布、油品品类占比、客单价趋势、支付方式分布、同比环比 |
| 价格管理 | 表单+历史 | 当前挂牌价、调价历史记录、促销价设置、竞品价格对比、发改委限价提示 |
| 会员管理 | 列表+详情 | 会员列表、积分/余额、消费记录、等级权益、充值活动配置 |
| 便利店管理 | 网格+库存 | 商品列表、库存管理、POS销售记录、临期预警、供应商管理 |
| 员工管理 | 表格+排班 | 员工花名册、排班日历、加油员业绩、交接班记录 |
| 设备管理 | 列表+状态 | 加油机状态、油罐检测仪、视频监控、洗车机、设备维保计划 |
| 安全合规 | 卡片+检查表 | 安全检查记录、消防设备检查、环保达标报告、证照到期提醒 |
| 连锁管理 | 多站对比+地图 | 各站业绩排名、统一调价、库存调配、区域经理看板 |
| 登录/权限 | 单栏居中 | 站长/员工/区域经理/总部角色、多站切换 |

**用户流程**: 查看今日销售→监控油品库存→调价/促销→管理会员→安排员工排班→检查安全合规
**类型差异**: 单站重日常运营; 连锁重跨站对比和统一管理; 综合能源站（油+电+氢）重多能源协调

### 11.6 物业管理（Property Management）
**适用类型**: 住宅小区物业、商业写字楼物业、产业园区物业、公寓管理

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 物业工作台 | 仪表盘+待办 | 今日待办工单数、报修统计、欠费提醒、通知横幅、快捷操作入口 |
| 业主/租户管理 | 列表+详情 | 住户信息（房号/姓名/联系方式/入住日期）、车辆信息、宠物登记、访客记录 |
| 房产管理 | 楼栋树+平面图 | 楼栋-单元-房间树形结构、房间状态（在住/空置/装修）、面积/户型信息、产权信息 |
| 收费管理 | 表格+账单 | 物业费/水电气/停车费账单列表、缴费状态、催缴记录、对公转账确认、电子发票 |
| 报修/工单 | 列表+流程 | 报修提交（拍照+描述+位置）、工单分配、处理进度跟踪、满意度评价、超时预警 |
| 巡检管理 | 路线+打卡 | 巡检路线设置、NFC/二维码打卡点、巡检记录（拍照+备注）、异常转工单 |
| 设备设施管理 | 台账+维保 | 电梯/消防/供配电/给排水设备台账、维保计划、维修记录、年检到期提醒 |
| 公告/通知 | 列表+推送 | 社区公告、停水停电通知、活动通知、个性化推送（按楼栋/单元） |
| 社区活动 | 卡片+报名 | 活动列表（图文）、在线报名、签到管理、活动回顾 |
| 访客/门禁 | 列表+二维码 | 访客预约、临时通行码生成、人脸识别管理、车辆进出记录 |
| 绿化/保洁 | 排班+检查表 | 保洁排班、绿化养护计划、卫生检查评分、消杀记录 |
| 投诉/建议 | 列表+处理流 | 投诉提交、分类标记、处理跟踪、回访确认、统计分析 |
| 财务报表 | 图表+表格 | 收入/支出统计、欠费率趋势、预算执行对比、月度/年度报告 |
| 登录/多角色 | 单栏居中 | 物业员工/经理/业主/租户角色、多项目切换 |

**用户流程**: 
- **物业员工**: 登录→工作台→处理报修工单→巡检打卡→催缴费用
- **业主**: 打开App→查看公告→提交报修→缴纳物业费→预约访客
**类型差异**: 住宅重社区活动和生活服务; 写字楼重租户管理和设备运维; 产业园重招商和企业服务; 公寓重租客流转和增值服务
