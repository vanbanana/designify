# 项目类型信息架构手册 — 物联网类

---

### 8.1 智能家居模式（Smart Home）
**适用类型**: 智能家居控制, 智能音箱管理, 智能照明, 智能安防

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 设备卡片网格+房间 | 房间Tab, 设备状态卡片(开关/温度/亮度), 场景快捷, 告警 |
| 设备详情 | 控制面板 | 开关/调节滑块, 定时, 联动规则, 设备信息, 固件更新 |
| 场景自动化 | 规则编辑 | 触发条件(时间/传感器/手动), 执行动作, 条件组合, 启用/禁用 |
| 家庭管理 | 房间分组 | 房间列表, 添加房间, 成员管理(权限), 家庭共享 |
| 设备添加 | 配网向导 | 选择设备类型, WiFi配网/蓝牙配对, 命名, 分配房间 |
| 消息/告警 | 列表 | 告警记录(异常/离线/低电量), 通知设置 |
| 能耗统计 | 图表 | 各设备能耗, 日/月趋势, 节能建议 |
| 个人中心 | 列表 | 家庭切换, 成员管理, 语音助手绑定, 设置 |

**用户流程**: 添加设备 → 配网 → 控制 → 设置自动化 → 查看状态 → 管理家庭
**类型差异**: 音箱重语音和音乐; 照明重色温和场景; 安防重摄像头和告警; 家电重状态和定时

---

### 8.2 工业设备管理模式（Industrial IoT）
**适用类型**: 工业IoT, 设备远程监控, 预测性维护, 产线监控

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 多面板仪表盘 | 设备总数/在线/告警, 产线状态, 关键指标, 地图分布 |
| 设备列表 | 表格+状态 | 设备名称, 类型, 状态(运行/停机/故障), 关键参数, 位置 |
| 设备详情 | 图表+参数 | 实时参数曲线, 历史趋势, 告警记录, 运维记录, 远程操作 |
| 告警中心 | 列表+筛选 | 告警列表(级别/时间/设备), 确认/处理, 告警规则配置 |
| 维护管理 | 日历+列表 | 维护计划, 工单, 备件库存, 维护记录 |
| 报表 | 图表 | OEE, 故障率, 能耗, 产能, 导出 |

**用户流程**: 监控仪表盘 → 发现告警 → 查看详情 → 远程操作/派工单 → 维护 → 恢复
**类型差异**: 远程监控重实时数据; 预测性维护重模型和趋势; 产线重OEE和瓶颈

---

### 8.3 可穿戴设备模式（Wearable Devices）
**适用类型**: 智能手表, 健康手环, AR眼镜, 智能戒指

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 数据摘要环 | 今日步数/心率/睡眠, 目标完成环, 快捷功能 |
| 运动记录 | 列表+地图 | 运动类型, GPS轨迹, 配速/心率曲线, 分段数据 |
| 健康监测 | 图表 | 心率趋势, 血氧, 睡眠分析, 压力, 体温 |
| 设备管理 | 列表 | 表盘/样式, 通知设置, 应用管理, 固件更新 |
| 个人中心 | 数据+列表 | 身体数据, 目标设置, 好友/挑战, 健康报告, 设置 |

**用户流程**: 佩戴设备 → 自动记录 → 查看数据 → 设定目标 → 运动训练 → 健康分析
**类型差异**: 手表重通知和应用; 手环重轻便和续航; AR眼镜重叠加和手势; 戒指重无感监测

---

### 8.4 智慧农业模式（Smart Agriculture）
**适用类型**: 农场管理, 作物监测, 农产品溯源, 农机管理, 温室控制

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 地图+数据 | 农场地图, 传感器数据(温湿度/土壤/光照), 告警, 天气预报 |
| 地块管理 | 地图+列表 | 地块划分, 作物信息, 种植计划, 生长阶段 |
| 监测数据 | 图表+地图 | 传感器实时数据, 历史趋势, 热力图, 异常标记 |
| 农机管理 | 列表+地图 | 农机状态, 作业记录, 调度, 维护计划 |
| 溯源查询 | 单栏+时间线 | 扫码查询, 种植/加工/运输全流程, 认证信息 |
| 报表 | 图表 | 产量, 成本, 质量, 对比分析, 导出 |

**用户流程**: 查看仪表盘 → 发现异常 → 查看详情 → 调度农机/灌溉 → 收获 → 溯源
**类型差异**: 农场重地块和计划; 溯源重全流程和认证; 农机重调度和油耗; 温室重环控

---

### 8.5 智慧工地模式（Smart Construction Site）
**适用类型**: 工地安全监控, 人员考勤, 环境监测, 塔吊监控

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 多面板 | 在场人数, 安全告警, 环境数据, 视频监控缩略图, 进度 |
| 人员管理 | 列表+统计 | 实名制考勤, 工种分布, 出勤统计, 培训记录 |
| 安全监控 | 视频网格+告警 | 实时视频, AI识别(安全帽/反光衣/越界), 告警列表 |
| 环境监测 | 图表 | PM2.5/噪声/风速, 超标告警, 历史趋势 |
| 设备监控 | 列表+参数 | 塔吊/升降机状态, 载重/风速/角度, 告警 |
| 进度管理 | 甘特图 | 施工进度, 里程碑, 延期预警 |

**用户流程**: 查看仪表盘 → 处理告警 → 检查监控 → 管理考勤 → 跟踪进度
**类型差异**: 安全重AI识别和告警; 考勤重实名和统计; 环境重达标和预警; 设备重参数和安全

---

### 8.6 环境监测模式（Environmental Monitoring）
**适用类型**: 空气质量监测, 水质监测, 噪声监测, 土壤监测

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 地图+数据 | 监测站分布, 各站数据, 超标标记, 整体评级 |
| 站点详情 | 图表+参数 | 实时数据, 历史趋势, 达标率, 各指标明细 |
| 报表 | 图表+表格 | 日报/月报/年报, 达标分析, 对比, 导出 |
| 告警 | 列表 | 超标告警, 设备故障, 处理记录 |
| 配置 | 表单 | 监测站管理, 阈值设置, 报表配置 |

**用户流程**: 查看仪表盘 → 发现超标 → 查看详情 → 生成报表 → 处理告警
**类型差异**: 空气重AQI和PM; 水质重COD和重金属; 噪声重分贝和时段; 土壤重污染因子

---

### 8.7 充电桩运营模式（EV Charging）
**适用类型**: 充电桩查找, 预约充电, 计费管理, 充电站运营

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 地图找桩 | 全屏地图+列表 | 附近充电站, 可用桩数, 距离, 价格, 筛选(快充/慢充) |
| 充电站详情 | 图文+列表 | 站点信息, 桩列表(状态/功率/价格), 导航, 评价 |
| 充电中 | 全屏数据 | 充电进度, 功率曲线, 已充/费用, 预计时间, 停止按钮 |
| 充电记录 | 列表 | 历史充电, 费用, 电量, 碳减排 |
| 运营后台 | 仪表盘 | 站点收入, 利用率, 桩状态, 故障告警, 电价策略 |

**用户流程**: 查找充电站 → 导航前往 → 扫码/插枪 → 充电 → 支付 → 评价
**类型差异**: 用户端重查找和支付; 运营端重利用率和收入; 快充重功率和等待

---

### 8.8 光伏/能源模式（Solar & Energy）
**适用类型**: 光伏监控, 能耗管理, 智能电网, 储能管理

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 数据+图表 | 发电量/用电量, 收益, 设备状态, 天气, 碳减排 |
| 发电详情 | 图表 | 日/月/年发电量, 功率曲线, 效率分析, 对比 |
| 能耗分析 | 图表+列表 | 各回路用电, 峰谷分析, 节能建议, 同比/环比 |
| 设备管理 | 列表+地图 | 逆变器/电池板状态, 故障告警, 运维记录 |
| 储能管理 | 图表+控制 | 电池SOC, 充放电策略, 电价, 收益优化 |
| 报表 | 图表 | 发电报表, 收益报表, 碳减排报告, 导出 |

**用户流程**: 查看发电 → 分析效率 → 管理能耗 → 处理故障 → 查看收益
**类型差异**: 光伏重发电和效率; 能耗重分析和节能; 智能电网重调度和负荷; 储能重策略和收益

---

### 8.9 车联网模式（Connected Vehicle）
**适用类型**: 车辆远程, 行车记录, 车况检测, OTA升级

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 车辆状态卡片 | 车辆位置, 油量/电量, 里程, 车门/车窗状态, 快捷控制 |
| 远程控制 | 按钮面板 | 解锁/锁车, 启动空调, 鸣笛/寻车, 后备箱 |
| 行车记录 | 列表+地图 | 行程记录, 油耗/电耗, 驾驶评分, GPS轨迹 |
| 车况检测 | 仪表盘 | 胎压, 电池, 发动机/电机, 保养提醒, 故障码 |
| OTA升级 | 单栏 | 版本信息, 更新内容, 下载, 安装, 进度 |
| 个人中心 | 列表 | 车辆管理, 驾驶员, 保养记录, 保险, 设置 |

**用户流程**: 查看车况 → 远程控制 → 行车 → 查看数据 → 保养/升级
**类型差异**: 电动车重充电和续航; 燃油车重油耗和保养; 商用车重车队和调度

---

### 8.10 宠物智能设备模式（Pet IoT）
**适用类型**: 智能喂食器, 宠物定位器, 宠物摄像头, 智能猫砂盆

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 设备卡片 | 设备状态, 宠物今日数据(喂食/活动/如厕), 告警 |
| 喂食器控制 | 表单+记录 | 喂食计划, 手动喂食, 食量设置, 喂食记录, 余粮提醒 |
| 定位追踪 | 地图+轨迹 | 实时位置, 活动轨迹, 围栏设置, 历史轨迹 |
| 摄像头 | 视频+控制 | 实时视频, 双向语音, 抓拍, 回放, 夜视 |
| 健康数据 | 图表 | 活动量, 睡眠, 体重趋势, 如厕频率 |
| 个人中心 | 列表 | 宠物档案, 设备管理, 数据导出, 设置 |

**用户流程**: 设置设备 → 自动运行 → 查看数据 → 远程交互 → 健康管理
**类型差异**: 喂食器重计划和余粮; 定位重围栏和轨迹; 摄像头重实时和回放; 猫砂盆重健康数据
