# 项目类型信息架构手册 — 新兴数字类

---

### 9.1 区块链浏览器模式（Blockchain Explorer）
**适用类型**: 区块链浏览器, 链上数据分析, Token追踪

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 搜索框+最新数据 | 全局搜索(地址/交易/合约), 最新区块列表, 最新交易, 链上统计 |
| 区块详情 | 数据表格 | 区块号, 时间, 交易数, Gas使用, 矿工/验证者, 交易列表 |
| 交易详情 | 时间线+数据 | 交易哈希, 状态, 发送/接收地址, 金额, Gas费, 事件日志 |
| 地址查询 | 资产+交易 | 地址余额, Token列表, 交易历史, NFT持有, 标签 |
| 合约详情 | 代码+交互 | 合约源码(已验证), ABI, 读写函数, 事件, 交互面板 |
| Token列表 | 表格 | Token排名, 价格, 市值, 持有者数, 交易量 |
| 统计图表 | 多图表 | 出块时间, Gas价格趋势, 活跃地址, 交易量, TVL |

**用户流程**: 搜索地址/交易 → 查看详情 → 追踪关联 → 分析数据 → 导出
**类型差异**: 通用浏览器重搜索和全面; 链上分析重可视化和标签; Token追踪重价格和持仓

---

### 9.2 加密钱包模式（Crypto Wallet）
**适用类型**: 多链钱包, DApp浏览器, NFT钱包, 硬件钱包配套App

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 资产列表 | 总资产, 各链/Token余额, 价格变动, 快捷操作(收/发/换) |
| 收款 | 单栏居中 | 地址二维码, 地址文本, 复制, 选择链/Token |
| 转账 | 表单 | 接收地址, Token选择, 金额, Gas设置, 确认, 签名 |
| 交易记录 | 列表 | 收发记录, 状态(待确认/成功/失败), 详情跳转 |
| DApp浏览器 | 浏览器+钱包 | URL栏, 已连接DApp, 交易签名弹窗, 授权管理 |
| NFT画廊 | 网格 | NFT缩略图, 集合分组, 详情, 发送, 上架 |
| 设置 | 列表 | 钱包管理, 助记词备份, 网络切换, 安全设置, 连接硬件钱包 |

**用户流程**: 创建/导入钱包 → 接收资产 → 转账/交互DApp → 管理NFT → 安全备份
**类型差异**: 多链重跨链和Gas; DApp浏览器重连接和签名; NFT重展示和交易; 硬件钱包重安全

---

### 9.3 DeFi模式（Decentralized Finance）
**适用类型**: 去中心化交易所(DEX), 借贷平台, 流动性挖矿, 收益聚合器

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| DEX首页 | 交换面板 | 代币交换面板, 价格图表, 流动性池, 钱包连接, 滑点设置 |
| 交换页 | 单栏居中 | 从/至代币选择, 金额输入, 汇率, 价格影响, 路由, 确认交换 |
| 流动性 | 列表+表单 | 流动性池列表, APY, TVL, 添加/移除流动性, 我的头寸 |
| 借贷 | 双面板 | 存入/借出, 利率, 抵押率, 清算线, 我的头寸 |
| 收益聚合 | 卡片网格 | 策略列表, APY, TVL, 风险等级, 存入/取出 |
| 仪表盘 | 数据+图表 | 总资产, 收益统计, 头寸分布, 历史收益 |
| 治理 | 列表 | 提案列表, 投票, 我的投票, 治理代币 |

**用户流程**: 连接钱包 → 交换/提供流动性/借贷 → 赚取收益 → 提取 → 参与治理
**类型差异**: DEX重交换和流动性; 借贷重抵押和清算; 挖矿重APY和锁仓; 聚合器重策略和风险

---

### 9.4 DAO治理模式（DAO Governance）
**适用类型**: DAO投票, 提案管理, 国库管理, 去中心化组织

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 概览+提案 | DAO简介, 国库余额, 活跃提案, 成员数, 治理指标 |
| 提案列表 | 列表+状态 | 提案标题, 状态(讨论/投票/执行/已完成), 投票数, 截止时间 |
| 提案详情 | 图文+投票 | 提案描述, 讨论区, 投票面板(赞成/反对/弃权), 投票权重 |
| 创建提案 | 表单 | 标题, 描述(Markdown), 执行操作, 门槛检查, 提交 |
| 国库 | 仪表盘 | 资产列表, 收支明细, 预算分配, 历史交易 |
| 成员 | 列表 | 成员列表, 投票权重, 贡献度, 角色 |

**用户流程**: 查看提案 → 参与讨论 → 投票 → 提案通过 → 链上执行
**类型差异**: 投票重权重和执行; 提案重格式和门槛; 国库重透明和预算

---

### 9.5 元宇宙模式（Metaverse）
**适用类型**: 虚拟世界, 虚拟社交, 虚拟地产, 数字孪生城市

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 入口/大厅 | 3D场景 | 虚拟大厅, 传送门, 活动入口, 好友在线, 导航 |
| 虚拟空间 | 3D环境 | 场景渲染, 化身控制, 空间音频, 交互物体 |
| 化身/装备 | 3D预览+列表 | 化身定制, 服装/配饰, NFT装备, 商城 |
| 虚拟地产 | 地图+列表 | 地块地图, 可用/已售, 价格, 购买/租赁 |
| 活动 | 日历+3D入口 | 虚拟活动列表, 演唱会/展览/会议, 预约, 进入 |
| 个人中心 | 列表 | 我的空间, 化身, 好友, 交易记录, 设置 |

**用户流程**: 创建化身 → 进入空间 → 社交/探索 → 购买资产 → 参加活动 → 创造内容
**类型差异**: 社交重化身和互动; 地产重位置和价值; 活动重沉浸和参与; 创造重工具和资产

---

### 9.6 数字孪生模式（Digital Twin）
**适用类型**: 工厂数字孪生, 城市数字孪生, 建筑BIM, 设备仿真

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 3D总览 | 3D场景+数据叠加 | 数字孪生模型, 实时数据标注, 告警高亮, 视角控制 |
| 对象详情 | 面板+3D聚焦 | 选中对象参数, 历史数据, 预测分析, 操作面板 |
| 仿真模拟 | 参数+结果 | 场景设置, 参数调整, 运行仿真, 结果对比 |
| 数据面板 | 多图表 | 关键指标, 趋势, 异常检测, 预测 |
| 告警 | 3D标注+列表 | 3D空间中的告警标记, 告警列表, 处理操作 |

**用户流程**: 查看3D总览 → 发现异常 → 聚焦对象 → 分析数据 → 仿真验证 → 执行操作
**类型差异**: 工厂重产线和设备; 城市重交通和能耗; 建筑重结构和系统; 设备重参数和预测

---

### 9.7 低代码/无代码模式（Low-code/No-code）
**适用类型**: 可视化应用构建器, 网站构建器, 工作流引擎, 数据库构建器

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页/项目列表 | 网格/列表 | 我的应用, 模板库, 新建应用, 最近编辑 |
| 应用编辑器 | 拖拽画布+属性面板 | 组件库(左侧), 画布(中间), 属性面板(右侧), 预览, 发布 |
| 数据管理 | 表格+表单 | 数据表列表, 字段管理, 数据CRUD, 关联, 导入导出 |
| 工作流编辑器 | 流程图 | 节点(触发/条件/动作), 连线, 变量, 测试运行 |
| 模板库 | 网格+预览 | 模板分类, 模板预览, 一键使用, 自定义 |
| 发布/部署 | 单栏 | 域名设置, 环境(开发/测试/生产), 版本, 访问权限, 发布 |
| 个人中心 | 列表 | 我的应用, 用量/配额, 团队, 账单, 设置 |

**用户流程**: 选择模板/新建 → 拖拽搭建 → 配置数据 → 设计工作流 → 预览测试 → 发布
**类型差异**: 应用构建重组件和逻辑; 网站构建重页面和设计; 工作流重节点和自动化; 数据库重字段和关联

---

### 9.8 插件/扩展模式（Plugin & Extension）
**适用类型**: 浏览器插件, IDE插件, 小程序开发平台, API市场

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 商店首页 | 搜索+推荐 | 搜索框, 热门/推荐/新品, 分类, 排行榜 |
| 插件详情 | 图文+操作 | 插件介绍, 截图/视频, 功能列表, 评分/评论, 安装/添加按钮 |
| 安装/管理 | 列表 | 已安装插件, 启用/禁用, 更新, 卸载, 权限 |
| 开发者后台 | 仪表盘 | 我的插件, 安装量, 评分, 收入, 用户反馈 |
| 发布 | 分步表单 | 插件包上传, 描述, 截图, 权限声明, 定价, 提交审核 |
| 个人中心 | 列表 | 已安装, 收藏, 购买记录, 开发者账号, 设置 |

**用户流程**: 搜索/浏览 → 查看详情 → 安装 → 使用 → 评价 → 更新/卸载
**类型差异**: 浏览器插件重权限和安全; IDE重语言和集成; 小程序重生态和审核; API重文档和调用
