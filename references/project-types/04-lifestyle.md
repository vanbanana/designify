# 项目类型信息架构手册 — 生活服务类

---

### 4.1 预约/预订模式（Booking/Reservation）
**适用类型**: 酒店预订, 民宿, 餐厅预订, 医院挂号, 美容预约, 健身课程预约, 牙科/宠物医院预约

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 搜索主导布局 | 目的地/服务搜索, 日期选择器, 人数/类型选择, 热门推荐, 分类入口 |
| 列表页 | 地图+列表分屏 | 筛选排序栏, 结果卡片(图片+价格+评分), 地图视图切换, 距离排序 |
| 详情页 | 图文混排长页 | 图片画廊/轮播, 设施/服务标签, 地图定位, 评价摘要, 价格日历, 预订CTA |
| 日历选择 | 日历网格 | 日期选择器, 价格日历, 时间段选择, 可用状态标记 |
| 预订表单 | 分步单栏 | 预订人信息, 特殊要求, 确认信息, 支付 |
| 支付确认 | 单栏居中 | 订单摘要, 支付金额, 确认编号, 添加到日历 |
| 订单管理 | 列表+状态标签 | 订单卡片, 状态(待确认/已确认/已完成), 取消/修改按钮 |
| 评价页 | 表单 | 星级评分, 文字评价, 图片上传, 标签 |
| 个人中心 | 列表+宫格 | 历史订单, 收藏, 会员信息, 设置 |

**用户流程**: 搜索(目的地+日期) → 浏览列表 → 查看详情 → 选择日期/房型 → 填写信息 → 支付 → 确认订单 → 管理订单
**类型差异**: 酒店重地图搜索和房型对比; 餐厅重时段和人数; 医院重科室和医生; 美容重技师选择; 健身重课程表和教练

---

### 4.2 房产模式（Real Estate）
**适用类型**: 房产租房, 二手房, 新房, 长租公寓, 共享办公, 商业地产

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 搜索+分类入口 | 买房/租房Tab, 区域/地铁/价格筛选, 热门推荐, 地图找房入口 |
| 列表页 | 列表+地图分屏 | 房源卡片(封面图+价格+面积+户型), 筛选排序, 地图标记价格 |
| 房源详情 | 沉浸式图文长页 | VR/图片画廊, 户型图, 价格/首付计算, 房源信息表, 周边配套地图, 经纪人卡片, 在线咨询CTA |
| 房贷计算器 | 表单+结果 | 总价/首付/利率输入, 月供计算结果, 还款明细表 |
| 收藏对比 | 表格对比布局 | 多房源并排对比(价格/面积/单价/朝向/楼层) |
| 预约看房 | 表单+日历 | 选择时间, 联系人, 经纪人确认 |
| 经纪人详情 | 个人信息页 | 头像/简介, 擅长区域, 历史成交, 评价, 联系按钮 |
| 发布房源 | 分步表单 | 基本信息, 图片上传, 价格设定, 户型描述, 联系方式 |
| 个人中心 | 宫格/列表 | 收藏, 浏览历史, 我的房源, 设置 |

**用户流程**: 搜索/筛选 → 浏览列表 → 查看详情 → 收藏/对比 → 联系经纪人/预约看房 → 房贷计算 → 成交
**类型差异**: 租房重月租和押金; 二手房重产权和税费; 新房重楼盘和户型; 长租公寓重租期和配套; 共享办公重工位类型和会议室; 商业地产重面积和业态

### 4.3 医疗健康模式（Healthcare）
**适用类型**: 挂号平台, 在线问诊, 健康记录, 药房, 体检中心, 康复中心, 牙科诊所, 宠物医院, 中医馆

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 功能入口网格 | 预约挂号入口, 在线问诊入口, 健康数据摘要, 常见科室快捷入口 |
| 科室/医院列表 | 列表+筛选 | 科室/医院卡片, 评分, 距离, 专长标签 |
| 医生详情 | 图文信息页 | 医生头像/简介, 擅长标签, 排班日历, 患者评价, 预约按钮 |
| 在线问诊 | 聊天界面 | 消息气泡, 图片/报告上传, 语音消息, 处方卡片, 问诊小结 |
| 健康档案 | 时间线+卡片 | 就诊记录, 检查报告列表, 过敏史, 用药史 |
| 健康数据 | 仪表盘 | 趋势折线图(血压/血糖), 每日步数环, 健康评分, 异常提醒 |
| 用药提醒 | 列表+提醒 | 药品列表, 服用时间, 剂量, 库存提醒, 拍照识药 |
| 检查报告 | 文档+图表 | 检查项目, 指标数值, 正常范围对比, 趋势图, 医生建议 |
| 个人中心 | 列表 | 就诊人管理, 挂号记录, 问诊记录, 家庭成员, 设置 |

**用户流程**: 症状自查/选科室 → 选医生 → 查看排班 → 预约/挂号 → 到院就诊/在线问诊 → 查看报告 → 健康档案管理
**类型差异**: 牙科重排牙方案和种植体; 宠物医院重宠物档案和疫苗; 中医重辨证开方和药材; 体检重套餐对比和报告解读; 康复重训练计划

### 4.4 本地生活模式（Local Services）
**适用类型**: 本地生活(美团), 家政维修, 跑腿代办, 搬家服务, 洗衣服务, 回收平台

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 定位+分类入口+推荐 | 地址定位栏, 服务分类图标, 热门推荐, 附近服务, 促销标签 |
| 服务列表 | 筛选排序列表 | 服务商卡片(评分+价格+距离), 分类筛选, 排序(距离/评分/价格) |
| 服务详情 | 图文+评价 | 服务描述, 价格表, 服务商信息, 用户评价, 预约/下单按钮 |
| 下单页 | 表单 | 服务时间, 地址, 特殊要求, 优惠券, 价格明细, 提交 |
| 订单跟踪 | 状态+地图 | 实时状态(已接单/进行中/已完成), 服务人员位置, 联系按钮 |
| 订单历史 | 列表 | 历史订单, 状态, 再次下单, 评价 |
| 个人中心 | 宫格/列表 | 我的订单, 优惠券, 地址管理, 收藏, 设置 |

**用户流程**: 定位地址 → 浏览/搜索服务 → 查看详情 → 下单 → 等待服务 → 确认完成 → 支付 → 评价
**类型差异**: 家政重服务人员和评价; 跑腿重实时位置和速度; 搬家重物品清单和报价; 洗衣重衣物识别和取送; 回收重物品分类和估价

### 4.5 交通出行模式（Transportation）
**适用类型**: 打车(滴滴/Uber), 共享单车, 公交地铁, 导航, 拼车, 停车管理

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 全屏地图+底部输入 | 地图, 目的地搜索框, 当前位置, 常用地点(家/公司), 附近车辆 |
| 车型选择 | 底部弹出列表 | 车型列表(快车/专车/豪华), 预估价格, 等待时间, 优惠券 |
| 等待接单 | 全屏地图+动画 | 地图(附近车辆动画), 预计等待时间, 取消按钮 |
| 行程中 | 全屏地图+信息栏 | 实时路线, 预计到达时间, 司机信息, 紧急按钮, 分享行程 |
| 行程结束 | 底部弹出卡片 | 费用明细, 支付方式, 评分(1-5星), 小费, 评价文字 |
| 订单历史 | 列表 | 历史行程, 费用, 路线, 再次叫车 |
| 个人中心 | 列表+宫格 | 钱包/余额, 优惠券, 常用地址, 紧急联系人, 设置 |
| 钱包 | 列表+卡片 | 余额, 充值, 支付方式, 消费记录, 优惠券 |

**用户流程**: 输入目的地 → 选车型 → 等待接单 → 上车 → 行程 → 支付 → 评价
**类型差异**: 共享单车重扫码和停放; 公交地铁重线路查询和实时到站; 导航重路线规划和路况; 拼车重路线匹配和费用分摊; 停车重车位查找和无感支付

|------|----------|----------|

| 个人中心 | 列表 | 我的车, 保养记录, 违章查询, 保险, 设置 |

**用户流程**: 搜索/浏览 → 车型详情 → 对比 → 预约试驾 → 购车 → 车主服务
**类型差异**: 二手车重检测报告和过户; 车联网重远程控制和车况; 充电桩重地图查找和充电状态; 驾考重题库和学时; 保养重预约和记录

| 购物车 | 底部弹出/独立页 | 菜品列表(加减数量), 备注, 优惠券, 合计, 去结算 |

| 会员 | 卡片+列表 | 会员等级, 积分, 优惠券, 消费记录, 充值 |

**用户流程**: 浏览菜单 → 选菜加购 → 结算 → 支付 → 等待/跟踪 → 用餐/收餐 → 评价
**类型差异**: 咖啡DTC重品牌故事和订阅; 茶叶/酒庄重产地和品鉴; 营养配餐重热量计算和膳食计划; 餐厅官网重氛围和品牌
