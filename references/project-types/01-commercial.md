# 项目类型信息架构手册 — 商业交易类

## 一、商业交易类

### 1.1 电商模式（E-commerce）
**适用类型**: 综合电商, 垂直电商, 跨境电商, 订阅电商, DTC品牌官网, 数字商品销售, 二手交易, 二手拍卖, 社区团购, 众筹平台, 预售平台, 按需打印(POD), 农资电商

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | F型布局+Banner轮播 | 搜索框, 分类导航, 推荐商品网格, 促销活动入口, 个性化推荐 |
| 商品列表 | 网格/列表切换布局 | 筛选器, 排序控件, 商品卡片, 分页器, 快速预览 |
| 分类页 | 树形导航+网格布局 | 分类树, 面包屑, 商品网格, 分类Banner |
| 搜索页 | 单栏+自动补全 | 搜索框, 搜索历史, 热门搜索, 搜索结果列表, 筛选 |
| 商品详情 | 双栏/单栏布局 | 商品图片轮播, 价格, SKU选择器, 加购按钮, 商品描述, 评价 |
| 购物车 | 单栏布局 | 商品列表, 数量调整, 价格汇总, 删除, 去结算按钮 |
| 结算页 | 分步单栏布局 | 收货地址, 配送方式, 优惠券, 支付方式, 订单汇总 |
| 支付页 | 单栏居中布局 | 支付金额, 支付方式选择, 二维码/跳转, 倒计时 |
| 支付结果 | 单栏居中布局 | 支付状态, 订单号, 查看订单按钮, 继续购物按钮 |
| 订单列表 | 状态Tab+列表布局 | 状态Tab, 订单卡片, 订单状态, 操作按钮, 分页 |
| 订单详情 | 时间线+详情布局 | 订单状态时间线, 商品信息, 物流信息, 支付信息, 操作 |
| 用户中心 | 宫格/列表布局 | 用户信息, 订单入口, 收藏入口, 优惠券, 积分, 设置 |
| 收货地址 | 列表+表单布局 | 地址列表, 默认地址, 新增/编辑表单, 删除 |
| 退换货 | 表单+列表布局 | 退换货原因, 商品选择, 上传凭证, 进度查询 |
| 登录注册 | 单栏居中布局 | 登录表单, 注册表单, 第三方登录, 忘记密码 |
| 收藏 | 网格/列表布局 | 收藏商品列表, 价格变动, 加购按钮, 删除 |
| 评价 | 列表+表单布局 | 评价列表, 评分, 图片/视频, 发表评价表单 |
| 客服/帮助 | 双栏布局 | 常见问题, 在线客服入口, 帮助中心, 联系方式 |

**用户流程**: 浏览→搜索/分类→详情→加购→结算→支付→订单确认→跟踪→评价
**类型差异**: 综合电商强调分类和搜索; 垂直电商突出专业属性筛选; 跨境电商需多语言/货币/关税; 订阅电商重周期管理; DTC品牌重品牌故事; 数字商品无需物流; 二手交易重卖家认证; 社区团购重自提点; 众筹重项目进度; 预售重发货时间; POD重定制预览; 农资重规格参数

### 1.2 市场平台模式（Marketplace）
**适用类型**: 市场平台(闲鱼/Airbnb), 自由职业平台, 按需服务平台, 任务众包, 打赏/赞助平台, 创作者订阅(Patreon)

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 搜索主导布局 | 搜索框, 分类导航, 推荐列表, 地理位置, Banner |
| 搜索/浏览 | 地图+列表/网格布局 | 地图视图, 列表视图, 筛选器, 价格区间, 排序 |
| 商品/服务详情 | 双栏布局 | 图片/视频, 详情描述, 卖家信息, 价格, 预订/购买按钮, 评价 |
| 卖家主页 | 个人主页布局 | 卖家信息, 评分, 商品/服务列表, 评价, 认证标识 |
| 发布页 | 分步表单布局 | 基本信息, 图片上传, 定价, 描述, 预览, 发布 |
| 卖家工作台 | 仪表盘布局 | 数据概览, 订单管理, 收入统计, 消息通知, 商品管理 |
| 商品管理 | 表格布局 | 商品列表, 状态筛选, 编辑/下架/删除, 批量操作 |
| 买方订单 | 列表布局 | 订单列表, 状态筛选, 订单详情, 联系卖家, 评价 |
| 卖方订单 | 表格布局 | 订单列表, 状态管理, 买家信息, 发货/服务, 收款 |
| IM聊天 | 双栏聊天布局 | 会话列表, 聊天窗口, 消息输入, 商品卡片, 快捷回复 |
| 支付结算 | 单栏布局 | 支付方式, 金额确认, 平台担保, 支付确认 |
| 钱包/收入 | 仪表盘布局 | 余额, 收入明细, 提现, 交易记录, 账单 |
| 评价管理 | 列表布局 | 收到的评价, 发出的评价, 回复评价, 评分统计 |
| 收藏 | 网格/列表布局 | 收藏列表, 价格变动, 快速操作 |
| 个人中心 | 列表布局 | 个人信息, 认证状态, 账户安全, 设置 |
| 实名认证 | 分步表单布局 | 身份证上传, 人脸识别, 银行卡绑定, 审核状态 |
| 纠纷仲裁 | 详情+表单布局 | 纠纷详情, 证据上传, 申诉表单, 仲裁进度 |
| 登录注册 | 单栏布局 | 登录/注册表单, 第三方登录, 手机验证 |
| 分类频道 | 频道页布局 | 分类导航, 热门商品, 推荐服务, 活动入口 |
| 平台规则 | 文档布局 | 规则分类, 规则详情, 搜索, 常见问题 |

**用户流程**: 
- 买方: 搜索→浏览→详情→联系/下单→支付→确认→评价
- 卖方: 入驻→认证→发布→管理→接单→收款→提现

**类型差异**: 闲鱼重C2C交易和即时沟通; Airbnb重地图搜索和预订日历; 自由职业重作品集和技能标签; 按需服务重地理位置和即时匹配; 任务众包重任务发布和竞标; 打赏重创作者内容和打赏入口; 创作者订阅重内容层级和订阅管理

### 1.3 零售POS/会员模式（Retail POS/Membership）
**适用类型**: POS收银系统, 会员管理, 库存管理, 便利店App, 自动售货机管理, 无人零售, 共享设备管理(充电宝/储物)

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| POS收银台 | 双栏布局(左商品右结算) | 商品搜索, 扫码输入, 商品列表, 数量调整, 折扣, 支付方式, 找零 |
| 商品管理 | 表格布局 | 商品列表, 分类筛选, 条码管理, 价格设置, 库存预警 |
| 库存管理 | 表格+仪表盘布局 | 库存列表, 入库/出库, 库存预警, 盘点, 调拨 |
| 会员管理 | 表格+详情布局 | 会员列表, 等级筛选, 会员详情, 积分, 消费记录 |
| 会员注册 | 表单布局 | 基本信息, 手机号, 会员等级, 开卡, 积分规则 |
| 交易记录 | 表格布局 | 交易列表, 时间筛选, 支付方式, 退款, 导出 |
| 交班结算 | 单栏布局 | 当班统计, 现金/扫码/卡支付, 差异, 交班确认 |
| 数据报表 | 仪表盘布局 | 销售趋势, 品类占比, 客单价, 时段分析, 同比环比 |
| 设备管理 | 卡片/地图布局 | 设备列表, 状态监控, 故障报警, 远程控制 |
| 共享设备 | 地图+列表布局 | 设备分布地图, 可用数量, 扫码使用, 归还点 |
| 用户端(小程序) | 单栏布局 | 扫码借用, 使用状态, 计费规则, 归还地图 |
| 优惠券管理 | 表格+表单布局 | 优惠券列表, 创建优惠券, 发放规则, 核销统计 |
| 供应商管理 | 表格布局 | 供应商列表, 联系方式, 供货记录, 结算 |
| 系统设置 | 分组表单布局 | 门店信息, 税率设置, 打印设置, 权限管理 |
| 员工管理 | 表格+权限布局 | 员工列表, 角色权限, 排班, 提成 |

**用户流程**: 
- 收银: 扫码/搜索→商品录入→折扣/优惠→支付→打印小票
- 会员: 注册→消费积分→等级提升→权益兑换
- 共享设备: 扫码→借用→使用→归还→自动扣费

**类型差异**: POS重收银效率和离线能力; 会员管理重积分和等级体系; 库存重预警和盘点; 便利店App重快速复购; 自动售货机重设备监控和补货; 无人零售重身份验证和自动结算; 共享设备重地理位置和计费规则

| 搜索 | 单栏布局 | 搜索框, 历史记录, 热门搜索, 搜索结果 |

**类型差异**: 美团团购重本地生活和到店核销; 社区团购重团长和自提点管理; 拼团重社交裂变和分享邀请; 美团重商家入驻和优惠券; 社区团购重预售和次日达; 拼团重人数门槛和虚拟成团

**类型差异**: 外卖平台重多商家和骑手调度; 点餐小程序重到店扫码和堂食; 自营外卖重品牌展示和会员体系; 外卖平台重配送时效; 点餐重桌台和排队; 自营重私域流量

**用户流程**: 
- DEX: 连接钱包→选代币→输入金额→确认交换→签名
- 借贷: 连接→抵押资产→借出→还款→解押
- 质押: 选池→质押→锁定期→获取收益→解押
- 治理: 查看提案→讨论→投票→执行

**类型差异**: DEX重流动性和滑点; 借贷重抵押率和清算; 质押重锁定期和APY; DAO重投票和提案; DEX重AMM; 借贷重风控; 质押重共识; DAO重去中心化决策
