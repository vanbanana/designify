# 项目类型信息架构手册 — 娱乐游戏类

---

### 6.1 游戏模式（Gaming）
**适用类型**: 游戏官网, 游戏社区, 游戏商城, 电竞平台

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 沉浸式Hero+分区 | 视频/动画Banner, 游戏特色, 下载CTA, 新闻, 社区入口 |
| 游戏介绍 | 图文长页 | 世界观, 角色/职业, 玩法, 系统要求, 截图/视频, 下载 |
| 新闻/公告 | 列表 | 更新日志, 活动公告, 版本说明 |
| 社区/论坛 | 信息流+板块 | 帖子列表, 发帖, 攻略, 截图分享, 讨论 |
| 商城 | 网格 | 道具/皮肤/礼包, 价格, 限定标记, 购物车 |
| 排行榜 | 表格 | 排名, 玩家名, 分数/段位, 赛季 |
| 赛事 | 列表+详情 | 赛程表, 对阵, 直播入口, 回放, 竞猜 |
| 下载页 | 单栏居中 | 平台选择(PC/移动/主机), 下载按钮, 配置要求 |
| 个人中心 | 仪表盘 | 战绩, 成就, 好友, 游戏库, 设置 |

**用户流程**: 了解游戏 → 下载 → 游玩 → 社区互动 → 购买道具 → 参加赛事
**类型差异**: 官网重品牌和下载; 社区重UGC和互动; 商城重道具和限定; 电竞重赛事和直播

---

### 6.2 流媒体模式（Streaming）
**适用类型**: 音乐流媒体(Spotify), 视频流媒体(Netflix), 在线KTV

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 横向滚动卡片行 | Banner推荐, "为你推荐"行, "最近播放"行, "排行榜"入口, 个性化推荐 |
| 搜索页 | 搜索栏+分类网格 | 搜索输入(热搜推荐), 分类标签, 搜索结果列表/网格 |
| 播放详情(音乐) | 沉浸式全屏 | 专辑封面(旋转动画), 进度条, 播放控制, 歌词滚动, 音质选择 |
| 播放详情(视频) | 沉浸式全屏 | 视频播放器, 弹幕开关, 选集列表, 评论列表, 相关推荐 |
| 播放列表/专辑 | 列表+封面 | 歌曲/剧集列表, 排序, 收藏, 下载 |
| 个人收藏 | Tab+网格/列表 | 我喜欢的/收藏歌单/下载/最近播放, 创建歌单按钮 |
| 下载管理 | 列表 | 已下载内容, 存储占用, 清理 |
| 设置 | 列表 | 音质/画质, 缓存, 通知, 账号, 隐私 |

**用户流程**: 浏览推荐/搜索 → 选择歌曲/视频 → 播放 → 收藏/加入歌单 → 分享 → 个人库管理
**类型差异**: 音乐重歌单和歌词; 视频重选集和弹幕; KTV重点歌和评分

---

### 6.3 桌游/剧本杀模式（Board Games & Escape Rooms）
**适用类型**: 桌游剧本杀, 密室逃脱预约, 棋牌室

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 主题推荐+分类 | 热门主题/剧本, 分类(恐怖/推理/欢乐), 难度, 人数 |
| 主题/剧本详情 | 图文+评价 | 主题介绍, 难度/时长/人数, 场景预览, 玩家评价, 预约 |
| 预约 | 日历+表单 | 选择日期/时段, 人数, 联系方式, 支付 |
| 个人中心 | 列表 | 预约记录, 评价, 收藏, 积分 |

**用户流程**: 浏览主题 → 查看详情 → 选择时段 → 预约 → 到店体验 → 评价
**类型差异**: 剧本杀重角色和剧情; 密室重难度和机关; 棋牌重计费

---

### 6.4 票务/活动模式（Ticketing & Events）
**适用类型**: 演出票务, 体育赛事, 展览, 会议报名

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 推荐+分类 | 热门演出/赛事, 城市选择, 分类(音乐/话剧/体育/展览), 日历视图 |
| 活动详情 | 图文+选座 | 活动介绍, 时间/地点, 座位图(可选区域), 价格, 购买 |
| 选座 | 交互式座位图 | 座位图, 价格分区, 已选/可选, 确认 |
| 订单 | 表单+支付 | 票种/数量, 联系人, 取票方式, 支付 |
| 电子票 | 卡片 | 二维码/条形码, 活动信息, 入场须知 |
| 个人中心 | 列表 | 我的票, 订单, 收藏, 设置 |

**用户流程**: 浏览活动 → 查看详情 → 选座/选票 → 支付 → 获取电子票 → 入场
**类型差异**: 演出重选座和档期; 赛事重对阵和座位; 展览重时间段; 会议重议程和签到

---

### 6.5 展会模式（Exhibition & Conference）
**适用类型**: 展会活动, 会议议程, 签到, 展商管理

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 展会介绍+入口 | 展会信息, 日程, 展商列表, 购票/注册, 地图 |
| 议程 | 时间轴+列表 | 日程表, 演讲/论坛, 时间/地点/嘉宾, 收藏, 提醒 |
| 展商列表 | 网格+详情 | 展商卡片, 分类, 展位地图, 联系, 产品 |
| 签到 | 单栏 | 二维码/条形码扫描, 电子胸卡, 议程同步 |
| 个人中心 | 列表 | 我的议程, 收藏展商, 名片交换, 笔记 |

**用户流程**: 注册/购票 → 查看议程 → 签到 → 参观/听会 → 交换名片 → 会后回顾
**类型差异**: 展会重展商和地图; 会议重议程和嘉宾; 签到重效率

---

### 6.6 拍卖模式（Auction）
**适用类型**: 拍卖行管理, 艺术品拍卖, 司法拍卖, 在线竞拍

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 推荐+分类 | 即将开拍, 热门推荐, 分类(艺术品/珠宝/房产/车辆), 日历 |
| 拍品详情 | 图文+竞价 | 拍品介绍, 图片, 估价, 当前出价, 出价历史, 倒计时, 出价按钮 |
| 出价 | 表单 | 输入出价, 确认, 自动出价(设定上限) |
| 我的竞拍 | 列表 | 在拍/已得/未得, 出价记录, 付款, 提货 |
| 专场 | 列表 | 专场介绍, 拍品列表, 预展时间 |

**用户流程**: 浏览拍品 → 预展/鉴定 → 注册 → 出价 → 竞得 → 付款 → 提货
**类型差异**: 艺术品重鉴定和来源; 司法拍卖重公告和合规; 在线重实时和自动出价

---

### 6.7 孵化器/商会模式（Incubator & Association）
**适用类型**: 孵化器管理, 商会协会, 投融资对接

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 入口+动态 | 入驻企业/会员列表, 活动, 新闻, 申请入驻 |
| 项目/企业 | 列表+详情 | 项目卡片, 行业/阶段, 团队, 融资需求, 联系 |
| 投融资对接 | 列表+匹配 | 投资人列表, 项目匹配, BP上传, 约见 |
| 活动 | 日历+列表 | 路演, 培训, 交流会, 报名 |
| 个人中心 | 列表 | 我的项目, 申请, 活动, 资源, 设置 |

**用户流程**: 申请入驻 → 展示项目 → 对接投资人 → 参加活动 → 获得资源
**类型差异**: 孵化器重项目培育; 商会重人脉和资源; 投融资重匹配和BP

### 6.8 数字藏品/元宇宙模式（NFT & Metaverse）
**适用类型**: 数字藏品, NFT交易市场, 元宇宙虚拟空间

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 推荐+搜索 | 热门藏品/系列, 新发行, 排行榜, 搜索 |
| 藏品详情 | 大图+信息 | 3D/图片预览, 创作者, 发行量, 价格, 拥有者, 购买/出价 |
| 市场 | 网格+筛选 | 在售藏品, 筛选(价格/稀有度/系列), 排序 |
| 我的藏品 | 网格 | 已收藏, 展示墙, 转售, 转账 |
| 发行/铸造 | 表单 | 上传作品, 设置属性, 定价, 发行量, 铸造 |
| 个人中心 | 列表 | 藏品库, 交易记录, 钱包, 设置 |

**用户流程**: 浏览 → 查看详情 → 购买/出价 → 持有/展示 → 转售
**类型差异**: NFT重链上交互和Gas; 数字藏品重合规和实名; 元宇宙重虚拟空间和交互

---

### 6.9 AR/VR模式（AR/VR Applications）
**适用类型**: AR应用, VR社交, VR游戏, 虚拟试穿

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 推荐+分类 | 推荐体验, 分类(游戏/社交/教育/工具), 新上架 |
| 体验详情 | 预览+操作 | 360°预览, 简介, 设备要求, 开始体验 |
| AR扫描 | 全屏相机+叠加 | 相机视图, AR叠加内容, 交互按钮, 保存 |
| VR空间 | 3D环境 | 虚拟场景, 化身, 空间音频, 交互 |
| 个人中心 | 列表 | 体验记录, 收藏, 设备管理, 设置 |

**用户流程**: 选择体验 → 检查设备 → 开始AR/VR → 交互 → 保存/分享
**类型差异**: AR重摄像头和叠加; VR重沉浸和控制器; 虚拟试穿重精确度

---

### 6.10 共享办公模式（Co-working Space）
**适用类型**: 共享办公(WeWork), 联合空间, 创客空间

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 搜索+推荐 | 附近空间, 热门空间, 价格, 设施 |
| 空间详情 | 图文+地图 | 环境照片, 位置, 设施(会议室/打印机/厨房), 价格, 预订 |
| 预订 | 日历+表单 | 工位类型(固定/流动/独立办公室), 日期, 时长, 支付 |
| 会议室预订 | 日历+网格 | 会议室列表, 容量, 设备, 时段选择, 预订 |
| 个人中心 | 列表 | 预订记录, 账单, 会员, 社区活动, 设置 |

**用户流程**: 搜索空间 → 查看详情 → 预订工位/会议室 → 入驻 → 续费/换座
**类型差异**: 共享办公重工位和会议室; 联合空间重社区和活动; 创客空间重设备和资源
