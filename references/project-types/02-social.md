# 项目类型信息架构手册 — 内容社交类

## 二、内容社交类

### 2.1 社交媒体模式（Social Media）
**适用类型**: 社交媒体(微信/微博/Instagram), 职场社交(LinkedIn), 校友社交, 匿名社区

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页/动态 | 信息流布局 | 动态列表, 发布入口, 故事/状态, 推荐内容, 下拉刷新 |
| 发布页 | 表单布局 | 文字输入, 图片/视频上传, 话题标签, 位置, 可见范围, 发布 |
| 消息 | 双栏聊天布局 | 会话列表, 聊天窗口, 消息输入, 表情/图片, 语音 |
| 个人主页 | 个人主页布局 | 头像/封面, 基本信息, 动态列表, 关注/粉丝, 相册 |
| 关注/粉丝 | 列表布局 | 用户列表, 关注/取关按钮, 互相关注标识 |
| 搜索 | 单栏布局 | 搜索框, 用户/内容/话题Tab, 搜索结果, 推荐 |
| 通知 | 列表布局 | 点赞/评论/关注通知, @我的, 系统通知 |
| 发现/探索 | 网格/卡片布局 | 热门话题, 推荐用户, 趋势内容, 分类导航 |
| 话题详情 | 详情+信息流布局 | 话题信息, 热度, 相关动态, 参与按钮 |
| 设置 | 列表布局 | 账户设置, 隐私设置, 通知设置, 黑名单, 注销 |
| 直播/故事 | 全屏布局 | 全屏视频, 互动评论, 礼物/点赞, 分享 |
| 相册 | 网格布局 | 照片网格, 分类相册, 上传, 编辑 |
| 收藏 | 列表布局 | 收藏内容, 分类管理, 取消收藏 |

**用户流程**: 注册→完善资料→发布动态→关注他人→浏览动态→互动(点赞/评论)→私信→发现更多
**类型差异**: 微信重私密社交和朋友圈; 微博重公开传播和热搜; Instagram重视觉和滤镜; LinkedIn重职业档案和招聘; 校友重学校认证; 匿名重隐私和话题

### 2.2 内容平台模式（Content Platform）
**适用类型**: 内容平台(知乎/Medium), 新闻聚合, 自媒体管理

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 信息流+分类布局 | 推荐内容, 分类Tab, 热门榜单, 搜索, 关注动态 |
| 内容详情 | 单栏阅读布局 | 标题, 作者信息, 正文, 点赞/评论/分享, 相关推荐 |
| 写作/发布 | 编辑器布局 | 富文本编辑器, 图片插入, 话题标签, 封面图, 预览, 发布 |
| 话题/专栏 | 列表布局 | 话题信息, 内容列表, 关注按钮, 热门内容 |
| 用户主页 | 个人主页布局 | 个人信息, 创作数据, 内容列表, 关注/粉丝, 专栏 |
| 搜索 | 单栏布局 | 搜索框, 内容/用户/话题筛选, 排序, 搜索结果 |
| 通知 | 列表布局 | 评论/点赞/关注通知, 系统通知 |
| 热榜 | 列表布局 | 热门内容排行, 热度指标, 上升/下降趋势 |
| 收藏夹 | 列表布局 | 收藏夹列表, 收藏内容, 分类管理 |
| 创作者中心 | 仪表盘布局 | 数据概览, 内容管理, 粉丝分析, 收益统计 |
| 内容管理 | 表格布局 | 已发布内容, 草稿, 审核中, 数据表现, 编辑/删除 |
| 评论管理 | 列表布局 | 评论内容, 回复, 删除, 屏蔽 |
| 设置 | 列表布局 | 账户设置, 内容偏好, 通知设置, 创作者设置 |

**用户流程**: 浏览→阅读→互动(点赞/评论/收藏)→关注→创作→发布→数据分析→收益
**类型差异**: 知乎重问答和专业性; Medium重阅读体验和订阅; 新闻聚合重时效和个性化推荐; 自媒体重数据分析和变现; 知乎重盐选专栏; Medium重会员阅读; 新闻重源头和事实核查

### 2.3 短视频/直播模式（Short Video/Live Streaming）
**适用类型**: 短视频(抖音), 直播(Twitch), 短视频+直播

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 推荐页 | 全屏沉浸式布局 | 全屏视频, 点赞/评论/分享, 作者信息, 音乐标识, 上下滑动 |
| 关注页 | 全屏布局 | 关注作者视频, 直播标识, 互动组件 |
| 拍摄/发布 | 相机界面布局 | 拍摄按钮, 滤镜/特效, 音乐选择, 剪辑, 发布 |
| 直播广场 | 网格布局 | 直播卡片, 在线人数, 分类标签, 推荐直播 |
| 直播间 | 全屏布局 | 视频画面, 弹幕评论, 礼物动画, 在线人数, 主播信息 |
| 直播互动 | 底部交互栏 | 礼物面板, 弹幕输入, 点赞动画, 分享, 关注 |
| 用户主页 | 个人主页布局 | 头像, 作品列表, 喜欢, 直播回放, 关注/粉丝 |
| 消息 | 列表布局 | 粉丝消息, 评论通知, @我的, 系统通知 |
| 搜索 | 单栏布局 | 搜索框, 用户/视频/直播/音乐Tab, 热搜 |
| 话题详情 | 详情+信息流布局 | 话题信息, 播放量, 相关视频, 参与拍摄 |
| 创作者中心 | 仪表盘布局 | 数据概览, 粉丝画像, 内容分析, 收益, 任务 |
| 钱包/收益 | 仪表盘布局 | 余额, 收入明细, 礼物收入, 提现 |
| 设置 | 列表布局 | 账户设置, 隐私设置, 通知设置, 青少年模式 |

**用户流程**: 
- 观众: 浏览推荐→观看→互动(点赞/评论/礼物)→关注→直播打赏
- 创作者: 拍摄/上传→编辑→发布→数据分析→收益提现

**类型差异**: 抖音重算法推荐和特效; Twitch重游戏直播和社区; 短视频重15秒吸引力; 直播重实时互动; 抖音重音乐和挑战赛; Twitch重订阅和Emote

### 2.4 论坛/社区模式（Forum/Community）
**适用类型**: 论坛(Reddit/V2EX), 问答平台(知乎/StackOverflow), Wiki百科

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 列表+侧栏布局 | 帖子列表, 热门/最新/精华Tab, 侧栏(热门话题/活跃用户) |
| 板块/节点 | 列表布局 | 板块信息, 帖子列表, 发帖按钮, 规则说明 |
| 帖子详情 | 详情+评论布局 | 标题, 正文, 作者, 点赞/收藏, 评论列表(嵌套), 回复 |
| 发帖/编辑 | 编辑器布局 | Markdown编辑器, 标题, 内容, 标签, 板块选择, 预览 |
| 问答首页 | 列表布局 | 问题列表, 未回答/热门/最新, 搜索, 我要提问 |
| 问题详情 | 详情+回答布局 | 问题描述, 标签, 回答列表, 投票排序, 采纳答案 |
| 回答 | 编辑器+列表布局 | 回答编辑器, 已有回答, 点赞/反对, 评论 |
| Wiki首页 | 分类+搜索布局 | 分类导航, 搜索框, 精选条目, 最近更改 |
| Wiki条目 | 文档布局 | 目录, 正文, 编辑历史, 讨论, 引用来源 |
| Wiki编辑 | 编辑器布局 | 富文本编辑器, 格式工具, 预览, 版本对比, 保存 |
| 用户主页 | 个人主页布局 | 个人信息, 发帖/回答/编辑记录, 声望/积分 |
| 消息 | 列表布局 | 私信, 评论通知, @我的, 系统通知 |
| 排行榜 | 列表布局 | 活跃用户, 高声望, 优质回答, 贡献排行 |
| 标签/话题 | 列表布局 | 标签列表, 相关帖子, 关注标签 |

**用户流程**: 浏览→阅读→回复/回答→发帖/提问→投票/采纳→积累声望→管理内容
**类型差异**: Reddit重投票和Subreddit; V2EX重节点和极客; 知乎重问答和专业; StackOverflow重代码和采纳; Wiki重协作和版本; Reddit重mod管理; V2EX重社区氛围; Wiki重中立性

**类型差异**: 个人博客重文字和SEO; 摄影社区重图片和EXIF; 作品集重项目展示; 博客重Markdown; 摄影重画廊; 作品集重案例叙述

**类型差异**: 播客平台重发现和订阅; 托管重上传和RSS; 有声书重章节和购买; 播客重免费和RSS; 托管重分发; 有声书重版权和付费

### 2.8 垂直社交模式（Vertical Social）
**适用类型**: 运动社交(Strava), 读书社交(Goodreads), 音乐社交, 情侣互动

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页(运动) | 信息流+数据布局 | 动态列表, 运动数据卡片, 排行榜, 挑战活动 |
| 记录运动 | 表单+地图布局 | 运动类型, GPS轨迹地图, 时长/距离/配速, 图片, 发布 |
| 运动详情 | 数据展示布局 | 轨迹地图, 数据图表(配速/心率/海拔), 分段数据, 评论 |
| 排行榜 | 表格布局 | 周/月/年排行, 分段排行, 好友排行 |
| 首页(读书) | 书架+信息流布局 | 在读/想读/已读书架, 动态, 推荐, 读书挑战 |
| 图书详情 | 详情+评论布局 | 封面, 简介, 作者, 评分, 书评, 版本, 想读/在读 |
| 书评 | 列表+表单布局 | 书评列表, 评分, 写书评, 短评 |
| 首页(音乐) | 播放器+社交布局 | 正在播放, 好友动态, 推荐歌单, 音乐品味 |
| 个人主页 | 个人主页布局 | 个人信息, 运动/阅读/音乐数据, 动态, 成就 |
| 挑战/活动 | 卡片布局 | 活动信息, 参与人数, 规则, 报名 |
| 好友 | 列表布局 | 好友列表, 关注, 粉丝, 推荐用户 |
| 消息 | 列表布局 | 私信, 互动通知, 活动通知 |
| 设置 | 列表布局 | 隐私设置, 通知设置, 数据同步, 账户 |

**用户流程**: 
- 运动: 记录运动→分享→好友互动→挑战→排行榜
- 读书: 标记图书→写书评→发现好书→读书挑战→好友推荐
- 音乐: 听歌→分享→发现音乐→好友品味对比

**类型差异**: Strava重GPS和运动数据; Goodreads重书评和书架; 音乐重播放记录和品味; 运动重数据追踪; 读书重阅读统计; 音乐重同步听歌
