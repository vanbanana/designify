# 项目类型信息架构手册 — 专业工具类

## 三、专业工具类

### 3.1 SaaS后台模式（SaaS Backend）
**适用类型**: SaaS后台, OA办公, 审批流, 报销系统, 考勤打卡, OKR管理

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 卡片+图表布局 | 数据卡片(KPI), 趋势图表, 快捷入口, 待办事项, 通知 |
| 数据分析 | 多栏图表布局 | 多维度图表, 时间筛选, 数据导出, 自定义看板 |
| 用户管理 | 表格+侧栏布局 | 用户列表, 搜索筛选, 用户详情侧栏, 批量操作 |
| 角色权限 | 矩阵布局 | 角色列表, 权限矩阵, 功能勾选, 数据权限 |
| 数据CRUD | 表格布局 | 数据列表, 新增/编辑/删除, 分页, 导入导出 |
| 数据详情 | 表单布局 | 表单字段, 校验规则, 关联数据, 保存/取消 |
| 项目管理 | 看板/甘特布局 | 任务卡片, 状态列, 拖拽排序, 甘特图, 里程碑 |
| 通知中心 | 列表布局 | 通知列表, 已读/未读, 分类筛选, 标记已读 |
| 审批流程 | 流程+表单布局 | 流程图, 审批节点, 表单填写, 通过/驳回, 历史 |
| 文件管理 | 网格/列表布局 | 文件列表, 上传, 预览, 下载, 分享, 权限 |
| 计费订阅 | 卡片布局 | 套餐对比, 当前套餐, 用量统计, 升级/续费 |
| 系统设置 | 分组表单布局 | 基础设置, 安全设置, 集成设置, 通知设置 |
| API管理 | 列表布局 | API密钥, 调用统计, 频率限制, 文档链接 |
| 操作日志 | 时间线布局 | 操作记录, 操作人, 时间, IP, 详情 |
| 登录注册 | 单栏布局 | 登录表单, SSO登录, 忘记密码, 企业注册 |
| 个人设置 | 表单布局 | 个人信息, 头像, 密码修改, 通知偏好 |

**用户流程**: 登录→仪表盘→查看数据→处理待办→管理用户/数据→审批→设置→退出
**Shell结构**: 顶部导航(Logo+搜索+通知+用户)+左侧菜单(可折叠)+面包屑+内容区
**类型差异**: SaaS后台重多租户和权限; OA重流程审批; 报销重票据和金额; 考勤重打卡和统计; OKR重目标对齐; SaaS重计费; OA重公文流转; 报销重财务集成

**类型差异**: 项目管理重甘特图和里程碑; 任务管理重清单和优先级; 敏捷看板重流动和迭代; 项目重范围; 任务重执行; 敏捷重响应变化

| 组织架构 | 树形图布局 | 组织树, 部门层级, 汇报关系, 拖拽调整 |
| 招聘管理 | 看板布局 | 职位列表, 候选人卡片, 阶段(简历/面试/Offer), 拖拽 |
| 职位发布 | 表单布局 | 职位信息, 要求, 薪资范围, 发布渠道, 提交 |
| 候选人详情 | 详情布局 | 简历, 面试记录, 评价, 状态, 操作 |

**用户流程**: 
- 招聘: 发布职位→筛选简历→面试→Offer→入职
- 员工: 入职→考勤→请假/加班→绩效→薪资→培训→离职
- 灵活用工: 发布需求→匹配人员→签到→完工→结算

**类型差异**: HR重全模块集成; 招聘重漏斗和体验; 猎头重候选人关系; 灵活用工重排班和结算; 薪资代发重合规; HR重组织发展; 招聘重雇主品牌; 灵活重即时性

**类型差异**: GitHub重协作和PR; API文档重交互示例; DevOps重流水线; 监控重实时指标; 日志重查询性能; 包管理重版本依赖
