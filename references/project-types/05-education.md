# 项目类型信息架构手册 — 教育知识类

---

### 5.1 在线教育模式（Online Education）
**适用类型**: 在线课程(Coursera/得到), MOOC, 技能培训, 编程学习, 语言学习

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 卡片网格+Banner轮播 | 推荐课程卡片, 分类导航, 搜索栏, 学习进度入口, 热门讲师 |
| 课程详情 | 上下长页(信息流) | 课程封面, 大纲折叠面板, 讲师信息, 评价列表, 价格/报名CTA, 试看入口 |
| 分类/搜索 | Tab+列表 | 分类树, 筛选(难度/时长/价格), 搜索结果, 排序 |
| 学习中心 | Tab+列表 | 课程进度条, 继续学习按钮, 已完成/进行中分类, 学习日历 |
| 视频课堂 | 沉浸式全屏 | 视频播放器, 章节列表, 笔记侧栏, 倍速/清晰度, 弹幕/评论 |
| 笔记/资料 | 列表+文档 | 课程资料列表, 下载, 笔记列表, 标签 |
| 作业/考试 | 表单布局 | 题目卡片, 计时器, 提交按钮, 成绩反馈, 错题本 |
| 个人中心 | 列表+数据 | 学习统计, 证书列表, 收藏课程, 学习时长, 设置 |
| 订单/支付 | 单栏 | 课程选择, 优惠券, 支付方式, 订单确认 |
| 社区/讨论 | 信息流 | 课程讨论区, 问答, 学习小组, 排行榜 |

**用户流程**: 浏览/搜索 → 课程详情 → 报名/支付 → 学习中心 → 视频课堂 → 完成作业 → 获得证书
**类型差异**: MOOC重大学合作和学分; 技能培训重实操和项目; 编程重代码编辑器和练习; 语言重口语和听力

---

### 5.2 知识付费模式（Knowledge Monetization）
**适用类型**: 付费专栏, 听书, 会员课程, 付费问答, 小报童

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 推荐+分类 | 精选专栏, 每日听书, 热门作者, 会员入口, 免费试听 |
| 专栏详情 | 图文+目录 | 作者信息, 专栏介绍, 目录列表(免费/付费标记), 价格, 订阅 |
| 内容阅读/收听 | 单栏阅读/播放器 | 正文排版, 音频播放器, 进度, 笔记, 划线, 分享 |
| 会员中心 | 卡片+列表 | 会员权益, 订阅管理, 专属内容, 续费 |
| 作者主页 | 个人品牌页 | 作者简介, 作品列表, 粉丝数, 关注 |
| 个人中心 | 列表 | 已购内容, 阅读/收听记录, 收藏, 笔记, 设置 |

**用户流程**: 发现内容 → 试听/试读 → 订阅/购买 → 持续学习 → 笔记/分享
**类型差异**: 专栏重深度和连载; 听书重音频和碎片化; 会员重权益和续费; 问答重互动和悬赏

---

### 5.3 考试/考证模式（Exam & Certification）
**适用类型**: 考试考证, 在线考试系统, 驾考学习, 题库App, 公务员/考研/CPA

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 考试入口+进度 | 选择考试, 学习进度, 今日任务, 排行榜, 倒计时 |
| 题库 | 分类+列表 | 章节列表, 题型(选择/填空/简答), 难度, 做题模式(练习/模拟) |
| 做题页 | 单栏聚焦 | 题目显示, 选项/输入, 计时器, 标记, 上一题/下一题, 交卷 |
| 成绩/解析 | 仪表盘+列表 | 总分, 各部分得分, 排名, 逐题解析, 错题标记 |
| 错题本 | 列表 | 错题列表, 正确率, 重做, 收藏 |
| 模拟考试 | 全屏 | 模拟真实考试环境, 计时, 自动交卷 |
| 个人中心 | 数据+列表 | 学习统计, 做题历史, 收藏, VIP管理, 设置 |

**用户流程**: 选择考试 → 刷题/看课 → 模拟考试 → 查看成绩 → 错题复习 → 参加考试
**类型差异**: 驾考重科目分类和学时; 公务员重行测/申论; 考研重政治/英语/专业课; CPA重多科切换

---

### 5.4 校园服务模式（Campus Services）
**适用类型**: 校园服务, 课程管理, 校园社交, 校园二手, 选课系统

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 功能入口网格 | 课表, 成绩, 选课, 校园卡, 通知, 二手市场 |
| 课表 | 日历网格 | 周视图, 课程色块, 教室/教师, 提醒 |
| 选课 | 列表+筛选 | 课程列表, 筛选(院系/时间/教师), 余量, 一键选课 |
| 成绩 | 表格 | 学期成绩, GPA, 排名, 趋势图 |
| 校园二手 | 网格/列表 | 发布/浏览二手物品, 分类, 价格, 联系 |
| 校园卡 | 卡片+记录 | 余额, 充值, 消费记录, 二维码支付 |
| 个人中心 | 列表 | 个人信息, 课程, 成绩, 社团, 设置 |

**用户流程**: 查课表 → 上课 → 查成绩 → 选课 → 校园消费 → 二手交易
**类型差异**: 课程管理重排课和冲突; 校园社交重校友关系; 二手重信息发布

---

### 5.5 阅读模式（Reading）
**适用类型**: 电子书(Kindle), 有声书(喜马拉雅), 漫画平台, 网文平台, 图书馆管理

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 推荐+书架 | 个性化推荐, 分类, 排行榜, 书架入口, 搜索 |
| 书架 | 网格 | 书籍封面网格, 阅读进度, 分类/排序, 下载管理 |
| 书籍详情 | 图文 | 封面, 简介, 目录, 评分, 评论, 开始阅读/购买 |
| 阅读器 | 沉浸式全屏 | 正文排版, 翻页/滚动, 字体/亮度调节, 书签/笔记, 进度 |
| 有声书播放 | 沉浸式 | 封面, 进度条, 播放控制, 倍速, 定时关闭, 章节列表 |
| 漫画阅读 | 全屏 | 页漫/条漫切换, 缩放, 章节列表, 追更 |
| 书城 | Tab+网格 | 分类, 排行榜, 限免, 新书, 搜索 |
| 个人中心 | 列表 | 阅读记录, 书架, 书签/笔记, 消费, VIP, 设置 |

**用户流程**: 发现书籍 → 加入书架 → 阅读/收听 → 书签/笔记 → 追更/续借 → 评论
**类型差异**: 电子书重排版和笔记; 有声书重播放和倍速; 漫画重阅读模式; 网文重追更和月票; 图书馆重检索和借阅

---

### 5.6 培训管理模式（Training Management）
**适用类型**: 培训机构, 驾校管理, 留学中介, 幼儿园管理, 员工培训(LMS)

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 仪表盘 | 数据卡片 | 学员数, 课程数, 收入, 排课率, 消课率 |
| 学员管理 | 表格+详情 | 学员列表, 档案, 课程进度, 缴费记录, 联系 |
| 排课管理 | 日历+表格 | 教师排班, 教室占用, 课程表, 冲突检测 |
| 消课/签到 | 列表+操作 | 课程列表, 学员签到, 课时扣减, 补课 |
| 家校互通 | 信息流+通知 | 学习报告, 作业通知, 家长留言, 公告 |
| 财务管理 | 表格+图表 | 收费记录, 退费, 课时消耗, 收入统计 |
| 系统设置 | 分组表单 | 机构信息, 教师管理, 教室管理, 课程设置 |

**用户流程**: 招生 → 报名缴费 → 排课 → 上课签到 → 消课 → 学习反馈 → 续费
**类型差异**: 驾校重学时和考试; 留学重申请和文书; 幼儿园重安全和家园; LMS重课程和学习路径

---

### 5.7 虚拟实验模式（Virtual Lab）
**适用类型**: 虚拟实验室, 仿真教学, 医学模拟

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 实验列表 | 实验分类, 推荐实验, 我的实验, 学习进度 |
| 实验详情 | 图文+操作 | 实验介绍, 目标, 原理, 步骤, 开始实验 |
| 实验操作 | 全屏交互 | 3D/2D实验环境, 工具栏, 操作指引, 数据记录, 保存 |
| 实验报告 | 文档 | 实验数据, 图表, 结论, 提交, 评分 |

**用户流程**: 选择实验 → 了解原理 → 操作实验 → 记录数据 → 生成报告 → 提交评分

### 5.8 图库素材模式（Asset Library）
**适用类型**: 图库素材站(Unsplash/Shutterstock), 模板市场, 字体站, 3D素材

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 搜索+瀑布流 | 大搜索框, 热门分类, 精选推荐, 趋势标签 |
| 搜索结果 | 瀑布流/网格 | 素材卡片, 筛选(类型/颜色/方向/尺寸), 分页 |
| 素材详情 | 大图+信息 | 预览大图, 分辨率/格式, 相似素材, 标签, 下载/购买 |
| 合集/分类 | 网格 | 主题合集, 分类浏览, 编辑推荐 |
| 上传(创作者) | 表单+批量 | 批量上传, 标签, 描述, 许可证, 收益 |
| 个人中心 | 列表 | 下载历史, 收藏, 许可证, 订阅, 上传管理 |

**用户流程**: 搜索/浏览 → 预览 → 下载/购买 → 使用 → 收藏
**类型差异**: 图库重搜索和分辨率; 模板重预览和编辑; 字体重试打和授权; 3D重格式和兼容性

---

### 5.9 博物馆/文化模式（Museum & Culture）
**适用类型**: 博物馆导览, 画廊展览, 文化馆, 艺术展览

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 展览推荐+入口 | 当前展览, 常设展览, 活动日历, 购票入口, 语音导览 |
| 展览详情 | 图文长页 | 展览介绍, 展品预览, 策展人语, 时间/地点, 购票 |
| 馆藏浏览 | 网格+筛选 | 藏品卡片, 分类(年代/类型/作者), 搜索, AR预览 |
| 语音导览 | 地图+音频 | 楼层地图, 语音点位, 自动播放, 文字说明 |
| 活动预约 | 日历+列表 | 讲座/工作坊, 时间, 名额, 预约 |
| 购票 | 表单 | 票种(成人/学生/团体), 日期, 数量, 支付 |

**用户流程**: 查看展览 → 购票 → 到馆 → 语音导览/AR → 参加活动 → 购买纪念品

---

### 5.10 志愿者/公益模式（Volunteer & Charity）
**适用类型**: 志愿者平台, 献血平台, 寻人启事, 公益捐赠

| 页面 | 布局模式 | 核心组件 |
|------|----------|----------|
| 首页 | 活动推荐+入口 | 志愿活动, 公益项目, 新闻动态, 志愿者故事 |
| 活动列表 | 卡片+筛选 | 活动卡片(时间/地点/类型), 筛选, 报名 |
| 活动详情 | 图文+报名 | 活动介绍, 时间地点, 需求人数, 服务内容, 报名按钮 |
| 我的服务 | 时间线+数据 | 服务记录, 工时统计, 证书, 积分 |
| 公益项目 | 详情+进度 | 项目介绍, 筹款进度, 捐助按钮, 项目动态 |
| 个人中心 | 列表 | 服务记录, 积分, 证书, 团队, 设置 |

**用户流程**: 浏览活动 → 报名 → 参加服务 → 记录工时 → 获得证书/积分
**类型差异**: 志愿者重活动和服务; 献血重点位和预约; 寻人重信息传播和线索; 捐赠重进度和透明
