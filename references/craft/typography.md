# Typography Craft — 排版工艺

> 通用排版规则，适用于任何品牌和风格。

## 字体配对原则

**一个项目最多 2 个字体族：**
- 1 个无衬线体（正文、UI 元素）
- 可选 1 个辅助字体（标题或代码）

**中文项目推荐字体栈：**
```css
font-family: "Inter", "PingFang SC", "Noto Sans SC", "Microsoft YaHei", system-ui, sans-serif;
```

**代码字体栈：**
```css
font-family: "JetBrains Mono", "Fira Code", "SF Mono", "Consolas", monospace;
```

## 字号层级（最小 5 级，最大 8 级）

| 角色 | 桌面端 | 移动端 | 用途 |
|------|--------|--------|------|
| Display | 48-72px | 32-48px | Hero 标题，极少使用 |
| H1 | 32-40px | 24-32px | 页面标题 |
| H2 | 24-28px | 20-24px | 区域标题 |
| H3 | 18-20px | 16-18px | 卡片/模块标题 |
| Body | 14-16px | 14-16px | 正文（中文不小于 14px） |
| Small | 12-13px | 12-13px | 辅助文字、标签 |
| Caption | 11-12px | 11-12px | 时间戳、脚注（最小值） |

## 字号三级体系（质量门禁强制检查）

### 等级 1: 正文/表单 — ≥ 14px（硬性阈值）
- 段落文字、输入框文字、按钮文字、表格内容
- 违反即返工，无例外
- 质量检查直接标记为"违规"

### 等级 2: 辅助标签/时间戳 — ≥ 12px（需注释标注）
- 仅限：标签（tag/badge）、时间戳、脚注、次要元数据
- **必须**在 CSS 中添加注释说明用途：`font-size: 12px; /* label */`
- 无注释标注的 12-13px 字号会被质量检查标记为"正文级违规"
- 不可用于关键操作标识或导航文字

### 等级 3: 装饰性小字 — ≥ 12px（条件允许）
- 仅限纯装饰性文字（版权信息、法律文本、极次要说明）
- **必须满足以下至少一项**：
  - 对比度 ≥ 4.5:1（比普通正文更高）
  - 字重 ≥ 500（加粗补偿）
- 必须添加注释：`font-size: 12px; /* decorative - high contrast */`

### 速查表

| 类型 | 最小字号 | 注释要求 | 补充要求 |
|------|----------|----------|----------|
| 正文/表单 | **14px** | 否 | — |
| 辅助标签 | **12px** | `/* label */` | — |
| 装饰性小字 | **12px** | `/* decorative */` | 对比度 ≥ 4.5:1 或 字重 ≥ 500 |

**绝对规则：** 中文正文 ≥ 14px，英文正文 ≥ 13px，任何文字 ≥ 11px（纯装饰文字最小 12px）

## 中文排版特殊规则

### 行高
- 正文：`line-height: 1.6` ~ `1.8`（中文比英文需要更多行间距）
- 标题：`line-height: 1.2` ~ `1.3`
- 段落间距：`margin-bottom: 1em` ~ `1.5em`

### 字间距
- 中文不要加 letter-spacing（中文字形自带间距）
- 英文全大写：`letter-spacing: 0.06em` ~ `0.1em`（必须加宽）
- 大标题（>32px）：`letter-spacing: -0.02em`（略微收紧）

### 中西混排
- 中英文之间自动加空格（盘古之白）："使用 React 开发" 而非 "使用React开发"
- 数字和中文之间同理："共 128 个页面"

## 字体权重规则

**一个项目最多使用 3 个权重：**
- 400 (Regular): 正文
- 500/600 (Medium/Semibold): 强调、标签、导航
- 700 (Bold): 标题

**常见错误：** 在一个页面里用 400/500/600/700/800 五个权重 → 视觉混乱

## 反模式

- ❌ 正文用 light (300) 权重 — 可读性差
- ❌ 标题和正文用同一个权重 — 层级不清
- ❌ 全大写中文（无意义，中文没有大小写）
- ❌ 正文用斜体中文（中文字体斜体效果差）
- ❌ 彩色正文（正文颜色应为灰黑色，不是彩色）

## 英文专用字体栈

```
英文场景（不含中文内容）：
"Inter", "Roboto", "SF Pro", system-ui, -apple-system, "Segoe UI", sans-serif
```

## 英文行高

```
英文正文行高：1.4-1.5（比中文紧凑，因为拉丁字母结构简单）
中文正文行高：1.6-1.75（汉字笔画复杂，需要更多呼吸空间）
```

## 最大行宽

```
正文最大行宽：65ch（约 45-75 个字符）
实现方式：max-width: 65ch;
超长行会导致读者视线回扫困难，降低阅读速度。
中文场景约等于 600-900px（14-16px 字号时）。
```
