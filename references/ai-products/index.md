## AI 产品通用交互范式

不管哪种 AI 产品，以下交互元素几乎都会出现：

### 输入层
- **提示词输入框** — 多行文本 + 快捷指令(/) + 附件上传
- **参数面板** — 温度/模型/Top-P 等高级设置（可折叠）
- **预设模板** — 常用提示词一键填充

### 输出层
- **流式输出** — 打字机效果 + 停止按钮 + 复制/重试/编辑按钮
- **Canvas/预览面板** — 输出物的实时预览（右侧或下方）
- **历史记录** — 过往生成结果的浏览和管理

### 过程层
- **Agent 步骤可视化** — 思考中 → 调用工具 → 生成结果
- **进度指示** — 长时间任务的进度条或阶段提示
- **中断/恢复** — 暂停生成 + 从断点继续

## 各模式核心页面

### 1. AI 对话（最复杂，18 个标准页面）
对话首页、新对话、对话历史、Agent 模式、Canvas 面板、
知识库管理、插件/工具市场、设置（模型/系统提示/温度）、
分享对话、团队协作、用量统计、API 管理、
多模态输入（图片/文件/语音）、上下文管理、记忆管理、
自定义 GPTs/Agent 创建、对话分支、导出

**核心布局：** 三栏式（侧边栏对话列表 + 中间对话流 + 右侧 Canvas/信息面板）

### 2. AI 图像生成（16 个标准页面）
生成首页、提示词编辑器、参数面板（尺寸/风格/模型/步数）、
生成结果网格、单图详情（放大/编辑/下载）、Inpainting 画布、
ControlNet 参考图、风格预设库、批量生成、生成历史、
模型选择/切换、社区画廊、收藏集、定价/额度、API 文档、设置

**核心布局：** 左侧参数面板 + 中间生成区域 + 右侧结果/历史

### 3. AI 工作流（15 个标准页面）
工作流编辑器（DAG 画布）、节点库、变量/环境管理、
运行日志/调试面板、版本历史、知识库管理、API 端点管理、
模板市场、团队协作、用量/计费、监控仪表盘、
Webhook 配置、子流程管理、权限管理、设置

**核心布局：** 全屏 DAG 画布 + 左侧节点面板 + 右侧属性面板 + 底部日志

### 4. AI 编程（15 个标准页面）
代码编辑器、Ghost Text 补全、Cmd+K 内联编辑、
Agent 模式（对话式编程）、文件树、终端面板、
Git 面板、问题面板、搜索、设置（模型/快捷键/主题）、
扩展市场、命令面板、项目切换、预览面板、文档

**核心布局：** IDE 三栏（文件树 + 编辑器 + 侧边面板）+ 底部终端 + 内联 AI

### 5. AI 视频生成（14 个标准页面）
生成首页（时间线编辑器）、提示词/脚本输入、参数面板（分辨率/帧率/时长/模型）、
生成预览（逐帧播放）、视频编辑器（裁剪/拼接/字幕）、风格预设库、
参考图/首帧上传、生成历史、批量生成队列、社区画廊、
模型选择/切换、定价/额度、API 文档、设置

**核心布局：** 上方时间线 + 中间预览区 + 下方参数面板 + 侧边历史/素材

### 6. AI 设计/画布（15 个标准页面）
画布编辑器（无限画布）、AI 生成面板（文生图/图生图）、
图层管理、组件/元素库、模板市场、品牌套件管理（Logo/字体/颜色）、
AI 辅助工具栏（智能抠图/背景移除/放大/修复）、协作面板（实时光标/评论）、
版本历史、导出（PNG/SVG/PDF/多尺寸批量）、资产管理、
团队空间、灵感画廊、定价/额度、设置

**核心布局：** 左侧工具栏 + 中间无限画布 + 右侧属性/AI面板 + 顶部操作栏

### 7. AI Agent/桌面助手（14 个标准页面）
主对话界面、任务规划面板（步骤拆解/进度）、工具调用可视化、
上下文管理器（当前加载的文件/数据/记忆）、工作流编辑器（自动化规则）、
知识库/记忆管理、插件/技能市场、多 Agent 协作面板、
任务历史/日志、用量统计、API/Webhook 配置、
权限管理（Agent 可访问范围）、个人偏好设置、系统设置

**核心布局：** 左侧对话/任务列表 + 中间主交互区 + 右侧上下文/工具面板

### 8. AI 数据分析（13 个标准页面）
数据上传/连接（CSV/数据库/API）、自然语言查询界面、
自动生成图表、仪表盘编辑器（拖拽图表组件）、数据表预览/筛选、
AI 洞察面板（自动发现趋势/异常/相关性）、报告生成器（AI 生成分析报告）、
数据集管理、查询历史、共享/发布、团队协作、
用量/计费、设置（数据源/权限/通知）

**核心布局：** 左侧数据源/数据集 + 中间查询+可视化区 + 右侧 AI 洞察面板

### 9. AI 音频（13 个标准页面）
生成首页（文本转语音/音乐生成）、输入面板（文本/歌词/描述）、
参数面板（音色/语言/情感/风格/BPM/乐器）、播放器（波形图+时间线）、
音频编辑器（裁剪/混音/降噪）、声音克隆（上传参考音频）、
音色库/预设库、批量生成、生成历史、项目管理、
定价/额度、API 文档、设置

**核心布局：** 上方输入/参数 + 中间波形播放器 + 下方项目/历史列表

### 10. AI 写作（14 个标准页面）
文档列表/工作区、富文本编辑器、AI 辅助侧栏（续写/改写/翻译/摘要/扩展）、
大纲/结构面板、AI 生成面板（指定风格/语气/长度）、模板库、
版本历史、协作编辑（评论/建议/实时光标）、发布管理（博客/公众号/多平台）、
SEO 优化面板（关键词建议/可读性评分）、资产管理（图片/引用/参考链接）、
写作统计（字数/阅读时间/产出趋势）、个人偏好（写作风格/常用模板）、设置

**核心布局：** 左侧大纲/导航 + 中间编辑器 + 右侧 AI 辅助面板 + 顶部工具栏

## 设计系统选择建议

| AI 产品模式 | 推荐风格 | 理由 |
|-------------|----------|------|
| AI 对话 | 简洁现代 或 暗色科技 | 对话界面需要低干扰，聚焦内容 |
| AI 图像生成 | 暗色科技 | 暗色背景让生成的图片更突出 |
| AI 视频生成 | 暗色科技 | 视频编辑需要沉浸式暗色环境，类似专业剪辑软件 |
| AI 设计/画布 | 简洁现代 或 活泼有趣 | 设计工具需要干净的画布环境，品牌感强 |
| AI 工作流 | 专业商务 | 工作流工具面向专业用户，需要信息密度 |
| AI 编程 | 暗色科技 | 开发者习惯暗色主题 |
| AI Agent/桌面 | 暗色科技 或 简洁现代 | Agent 工具偏技术向，需要展示大量状态信息 |
| AI 数据分析 | 专业商务 或 简洁现代 | 数据仪表盘需要清晰的信息层级和图表可读性 |
| AI 音频 | 暗色科技 | 音频工具参考 DAW（数字音频工作站）的暗色传统 |
| AI 写作 | 温暖文艺 或 简洁现代 | 写作工具需要舒适的阅读体验 |
