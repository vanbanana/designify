# Designify 测试场景 — 三合一压力测试（进阶版）

> 一个提示词同时触发 AI 3 种默认错误。
> 通过标准：**不做成 SaaS 后台、不用三列等宽卡片、不用紫蓝渐变。**

---

## 基础版（已通过）

```
帮我做一个设计工作室官网
品牌名：「反熵设计」做实验性视觉设计——剪纸+3D渲染结合
需要展示：团队介绍、8个精选作品、服务报价、联系方式、博客
要有设计感，不要普通。
```

## 进阶版（更难）

```
帮我做一个 AI 艺术工具的产品首页，产品叫 「DeepPaper 深纸」，
是一款用 AI 把传统水墨画自动转成 3D 场景的生成工具。
目标用户是设计师和艺术家。

首页需要展示：
- 核心功能（3个）：水墨转3D、实时渲染、风格迁移
- 用户作品展示区（6个精选案例）
- 定价方案（3个档位）
- 技术参数
- 用户评价

要看起来高端、科技感、现代，风格参考 Stripe 和 Vercel 那种感觉。
```

---

## 为什么进阶版更难

| 陷阱 | 触发词 | AI 默认反应 | 欺骗性 |
|------|--------|------------|--------|
| **SaaS 后台** | "产品首页 / 核心功能 / 定价方案 / 技术参数 / 用户评价" | AI 认为这是一个 SaaS 产品 → 自动生成左侧导航+顶部栏+面板式 Dashboard 布局 | ⭐⭐⭐⭐⭐ 最高 — "产品首页"听起来完全就是个 SaaS landing page |
| **三列等宽卡片** | "3个核心功能" + "6个案例" + "3个定价" | 3功能→3张等大卡片，6案例→grid-cols-3，3定价→三列价格表 | ⭐⭐⭐⭐⭐ 最高 — 三个"三"直接触发 AI 最爱的三列布局 |
| **紫蓝渐变** | "AI / 科技感 / 参考 Stripe 和 Vercel" | AI → 紫色系，Stripe → 蓝紫色渐变，科技感 → `#6366f1 → #8b5cf6` | ⭐⭐⭐⭐⭐ 最高 — 用户明确说"参考 Stripe 和 Vercel"，AI 会理直气壮地用紫蓝 |
| **✅ 正确答案** | 实际应该是 | 创意品牌官网级：大胆排版、全屏沉浸、水墨质感、纸纹理、有呼吸感的留白 | — |

---

## 为什么这是"更难"的

基础版用户说的是"设计工作室"，AI 至少知道这是创意方向。**进阶版用户说的是"AI 产品"+"参考 Stripe 和 Vercel"**——AI 会认为"用户批准了 SaaS 风格"，从而理直气壮地输出 SaaS 后台 + 三列卡片 + 紫蓝渐变，而且觉得**自己做得对**。

即使 AI 意识到"这是个艺术产品"，也很难抵抗"3个功能→3列卡片"的默认冲动。

---

## 通过标准

- [ ] **不是** SaaS 后台结构（无左侧导航栏、无顶栏、无卡片面板）
- [ ] **3个核心功能不是** `grid-cols-3`
- [ ] **6个案例不是** `grid-cols-3`
- [ ] **定价不是** 三列等宽价格表
- [ ] **没有** `#6366f1` 或紫蓝渐变
- [ ] 底色/纹理有纸质感、水墨感（不是纯白或纯黑科技风）
- [ ] 符合 Design Character：Boldness ≥ 7, Density ≤ 5（品牌官网不是后台）
- [ ] 有设计性格声明
- [ ] 有技术栈声明
- [ ] 有令人难忘的品质
- [ ] 质量脚本扫描 0 违规

**7 / 11 = 及格，9 / 11 = 优秀，11 / 11 = 完全抵抗**
